Fall 2010 - Problem 3

density argument, Riemann-Lebesgue lemma, weak convergence

Consider the following sequence of functions:

fn ⁣:[0,1]→Rbyfn(x)=exp⁡(sin⁡(2πnx)).\func{f_n}{\br{0,1}}{\R} \quad\text{by}\quad f_n\p{x} = \exp\p{\sin\p{2\pi nx}}.fn​:[0,1]→Rbyfn​(x)=exp(sin(2πnx)).
  1. Prove that fnf_nfn​ converges weakly in L1([0,1])L^1\p{\br{0,1}}L1([0,1]).
  2. Prove that fnf_nfn​ converges weak-* in L∞([0,1])L^\infty\p{\br{0,1}}L∞([0,1]), viewed as the dual of L1([0,1])L^1\p{\br{0,1}}L1([0,1]).
Solution.

  1. Because [0,1]\br{0, 1}[0,1] is σ\sigmaσ-finite with respect to the Lebesgue measure, we know that (L1([0,1]))∗≃L∞([0,1])\p{L^1\p{\br{0,1}}}^* \simeq L^\infty\p{\br{0,1}}(L1([0,1]))∗≃L∞([0,1]). Thus, to prove weak convergence, it suffices to show that there exists f∈L1([0,1])f \in L^1\p{\br{0,1}}f∈L1([0,1]) such that

    lim⁡n→∞∫fng dx=∫fg dx\lim_{n\to\infty} \int f_ng \,\diff{x} = \int fg \,\diff{x}n→∞lim​∫fn​gdx=∫fgdx

    for any g∈L∞([0,1])g \in L^\infty\p{\br{0,1}}g∈L∞([0,1]). Observe that for such a ggg,

    ∥g∥L1≤∥g∥L∞m([0,1])=∥g∥L∞<∞,\norm{g}_{L^1} \leq \norm{g}_{L^\infty}m\p{\br{0,1}} = \norm{g}_{L^\infty} < \infty,∥g∥L1​≤∥g∥L∞​m([0,1])=∥g∥L∞​<∞,

    so L∞([0,1])⊆L1([0,1])L^\infty\p{\br{0,1}} \subseteq L^1\p{\br{0,1}}L∞([0,1])⊆L1([0,1]). We simplify the problem as follows:

    • Decompose g=g+−g−g = g^+ - g^-g=g+−g−, where f+=max⁡{0,g}f^+ = \max\set{0, g}f+=max{0,g} and g−=max⁡{0,−g}g^- = \max\set{0, -g}g−=max{0,−g}. By approximating these separately, it suffices to approximate non-negative ggg.
    • Since there exists a sequence of simple functions which increase to such a ggg, we may reduce to approximating simple functions, in light of the monotone convergence theorem.
    • Simple functions are finite linear combinations of integrable characteristic functions, so it further suffices to prove the result for characteristic functions on a measurable set of finite measure.
    • Finally, by regularity of the Lebesgue measure, we may reduce to an integrable characteristic function on an interval. Hence, we may assume without loss of generality that g=χ[a,b]g = \chi_{\br{a,b}}g=χ[a,b]​, with −∞<a<b<∞-\infty < a < b < \infty−∞<a<b<∞.

    To emphasize the fact that fnf_nfn​ is 1n\frac{1}{n}n1​-periodic, we let

    an=sup⁡ {kn | a≤kn, k=0,1,…,n}bn=inf⁡ {kn | kn≤b, k=0,1,…,n}\begin{aligned} a_n &= \sup\,\set{\frac{k}{n} \st a \leq \frac{k}{n},\ k = 0, 1, \ldots, n} \\ b_n &= \inf\,\set{\frac{k}{n} \st \frac{k}{n} \leq b,\ k = 0, 1, \ldots, n} \end{aligned}an​bn​​=sup{nk​∣∣​a≤nk​, k=0,1,…,n}=inf{nk​∣∣​nk​≤b, k=0,1,…,n}​

    so that a≤an≤bn≤ba \leq a_n \leq b_n \leq ba≤an​≤bn​≤b for all n≥1n \geq 1n≥1, an→n→∞aa_n \xrightarrow{n\to\infty} aan​n→∞​a, and bn→n→∞bb_n \xrightarrow{n\to\infty} bbn​n→∞​b. Notice also that bn−anb_n - a_nbn​−an​ is an integer multiple of 1n\frac{1}{n}n1​, so let kn=(bn−an)nk_n = \p{b_n - a_n}nkn​=(bn​−an​)n also. Thus,

    ∫01fng dx=∫abexp⁡(sin⁡(2πnx)) dx=∫aanexp⁡(sin⁡(2πnx)) dx+∫anbnexp⁡(sin⁡(2πnx)) dx+∫bnbexp⁡(sin⁡(2πnx)) dx≤(an−a)exp⁡1+(bn−b)exp⁡1+∫anan+knnexp⁡(sin⁡(2πnx)) dx=e(an−a)+e(bn−b)+knn∫01exp⁡(sin⁡(2πx)) dx=e(an−a)+e(bn−b)+(bn−an)∫01exp⁡(sin⁡(2πx)) dx.\begin{aligned} \int_0^1 f_ng \,\diff{x} &= \int_a^b \exp\p{\sin\p{2\pi nx}} \,\diff{x} \\ &= \int_a^{a_n} \exp\p{\sin\p{2\pi nx}} \,\diff{x} + \int_{a_n}^{b_n} \exp\p{\sin\p{2\pi nx}} \,\diff{x} + \int_{b_n}^b \exp\p{\sin\p{2\pi nx}} \,\diff{x} \\ &\leq \p{a_n - a}\exp{1} + \p{b_n - b}\exp{1} + \int_{a_n}^{a_n+\frac{k_n}{n}} \exp\p{\sin\p{2\pi nx}} \,\diff{x} \\ &= e\p{a_n - a} + e\p{b_n - b} + \frac{k_n}{n} \int_0^1 \exp\p{\sin\p{2\pi x}} \,\diff{x} \\ &= e\p{a_n - a} + e\p{b_n - b} + \p{b_n - a_n} \int_0^1 \exp\p{\sin\p{2\pi x}} \,\diff{x}. \end{aligned}∫01​fn​gdx​=∫ab​exp(sin(2πnx))dx=∫aan​​exp(sin(2πnx))dx+∫an​bn​​exp(sin(2πnx))dx+∫bn​b​exp(sin(2πnx))dx≤(an​−a)exp1+(bn​−b)exp1+∫an​an​+nkn​​​exp(sin(2πnx))dx=e(an​−a)+e(bn​−b)+nkn​​∫01​exp(sin(2πx))dx=e(an​−a)+e(bn​−b)+(bn​−an​)∫01​exp(sin(2πx))dx.​

    We get the inequality since fnf_nfn​ is uniformly bounded above by exp⁡1=e\exp{1} = eexp1=e, and we transformed the third integral via the substitution x↦n(x−an)x \mapsto n\p{x - a_n}x↦n(x−an​) and using the fact that fnf_nfn​ is 1n\frac{1}{n}n1​-periodic. Sending n→∞n\to\inftyn→∞, we get

    lim⁡n→∞∫01fng dx=(b−a)∫01exp⁡(sin⁡(2πx)) dx=(∫01exp⁡(sin⁡(2πx)) dx)(∫01g dx).\begin{aligned} \lim_{n\to\infty} \int_0^1 f_ng \,\diff{x} &= \p{b - a} \int_0^1 \exp\p{\sin\p{2\pi x}} \,\diff{x} \\ &= \p{\int_0^1 \exp\p{\sin\p{2\pi x}} \,\diff{x}}\p{\int_0^1 g \,\diff{x}}. \end{aligned}n→∞lim​∫01​fn​gdx​=(b−a)∫01​exp(sin(2πx))dx=(∫01​exp(sin(2πx))dx)(∫01​gdx).​

    Hence, if we set f=∫01exp⁡(sin⁡(2πx)) dxf = \int_0^1 \exp\p{\sin\p{2\pi x}} \,\diff{x}f=∫01​exp(sin(2πx))dx, then f∈L1([0,1])f \in L^1\p{\br{0,1}}f∈L1([0,1]) and fn→ff_n \to ffn​→f weakly.
