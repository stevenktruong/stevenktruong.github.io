Spring 2018 - Problem 11

conformal mappings, Schwarz reflection principle

For R>1R > 1R>1, let ARA_RAR​ be the annulus {1<∣z∣<R}\set{1 < \abs{z} < R}{1<∣z∣<R}. Assume there is a conformal (i.e., injective, holomorphic) mapping FFF from AR1A_{R_1}AR1​​ onto AR2A_{R_2}AR2​​. Prove that R1=R2R_1 = R_2R1​=R2​.
Solution.

First, suppose FFF maps ∂D\partial\D∂D to ∂DR2\partial\D_{R_2}∂DR2​​. Then if we replace FFF with R2F\frac{R_2}{F}FR2​​, which is holomorphic since F(z)≠0F\p{z} \neq 0F(z)=0, we see that if ∣z∣=1\abs{z} = 1∣z∣=1, then

∣R2F(z)∣=R2R2=1.\abs{\frac{R_2}{F\p{z}}} = \frac{R_2}{R_2} = 1.∣∣​F(z)R2​​∣∣​=R2​R2​​=1.

Since 1z\frac{1}{z}z1​ is an automorphism of C∖{0}\C \setminus \set{0}C∖{0}, it follows that R2F\frac{R_2}{F}FR2​​ is still conformal. Hence, without loss of generality, we may assume that FFF maps ∂D\partial\D∂D to ∂D\partial\D∂D, and so FFF maps ∂DR1\partial\D_{R_1}∂DR1​​ to ∂DR2\partial\D_{R_2}∂DR2​​.

Let

G(z)={F(z)if 1≤∣z∣≤R1,1/F(1/z‾)‾if R1−1≤∣z∣≤1.G\p{z} = \begin{cases} F\p{z} & \text{if } 1 \leq \abs{z} \leq R_1, \\ \conj{1/F\p{1/\conj{z}}} & \text{if } R_1^{-1} \leq \abs{z} \leq 1. \end{cases}G(z)={F(z)1/F(1/z)​​if 1≤∣z∣≤R1​,if R1−1​≤∣z∣≤1.​

If ∣z∣=1\abs{z} = 1∣z∣=1, then because ∣F(z)∣=1\abs{F\p{z}} = 1∣F(z)∣=1 also, we get

1/F(1/z‾)‾=1/F(z)‾=F(z),\conj{1/F\p{1/\conj{z}}} = \conj{1/F\p{z}} = F\p{z},1/F(1/z)​=1/F(z)​=F(z),

so GGG is continuous at ∂D\partial\D∂D. By the Schwarz reflection principle, FFF extends holomorphically to a map F ⁣:AR1→D\func{F}{A_{R_1}}{\D}F:AR1​​→D such that if ∣z∣=R1−1\abs{z} = R_1^{-1}∣z∣=R1−1​, then

1∣z∣=R1  ⟹  ∣F(1/z‾)∣=R2  ⟹  ∣G(z)∣=1∣F(1/z‾)∣=R2−1.\frac{1}{\abs{z}} = R_1 \implies \abs{F\p{1/\conj{z}}} = R_2 \implies \abs{G\p{z}} = \frac{1}{\abs{F\p{1/\conj{z}}}} = R_2^{-1}.∣z∣1​=R1​⟹∣F(1/z)∣=R2​⟹∣G(z)∣=∣F(1/z)∣1​=R2−1​.

By induction, FFF extends to a holomorphic map F ⁣:D∖{0}→D\func{F}{\D \setminus \set{0}}{\D}F:D∖{0}→D such that if ∣z∣=R1−k\abs{z} = R_1^{-k}∣z∣=R1−k​, then ∣F(z)∣=R2−k\abs{F\p{z}} = R_2^{-k}∣F(z)∣=R2−k​ for infinitely many k∈Nk \in \Nk∈N. Since FFF is bounded near 000, we see that 000 is a removable singularity, and because R1−k↦R2−kR_1^{-k} \mapsto R_2^{-k}R1−k​↦R2−k​, we see that F(0)=0F\p{0} = 0F(0)=0. ∣F(z)∣≤∣z∣  ⟹  R2−k≤R1−k  ⟹  R1≤R2.\abs{F\p{z}} \leq \abs{z} \implies R_2^{-k} \leq R_1^{-k} \implies R_1 \leq R_2.∣F(z)∣≤∣z∣⟹R2−k​≤R1−k​⟹R1​≤R2​.

Running the same argument with F−1F^{-1}F−1, we see that R1=R2R_1 = R_2R1​=R2​, by symmetry.
