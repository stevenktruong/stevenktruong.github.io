Spring 2013 - Problem 8

Hadamard's three-lines theorem, operator theory

Let AAA and BBB be positive definite n×nn \times nn×n real symmetric matrices with the property

∥BA−1x∥≤∥x∥for all x∈Rn\norm{BA^{-1}x} \leq \norm{x} \quad\text{for all } x \in \R^n∥∥​BA−1x∥∥​≤∥x∥for all x∈Rn

where ∥x∥\norm{x}∥x∥ denotes the usual Euclidean norm: ∥x∥2=∑i=1n∣xi∣2\norm{x}^2 = \sum_{i=1}^n \abs{x_i}^2∥x∥2=∑i=1n​∣xi​∣2.

  1. Show that for each pair x,y∈Rnx, y \in \R^nx,y∈Rn,

    z↦⟨y,BzA−zx⟩z \mapsto \inner{y, B^zA^{-z}x}z↦⟨y,BzA−zx⟩

    admits an analytic continuation from 0<z<10 < z < 10<z<1 to the whole complex plane. Here ⟨ ⋅ , ⋅ ⟩\inner{\:\cdot\:, \:\cdot\:}⟨⋅,⋅⟩ denotes the usual Euclidean inner product on Rn\R^nRn.

  2. Show that

    ∥BθA−θx∥≤∥x∥\norm{B^\theta A^{-\theta} x} \leq \norm{x}∥∥​BθA−θx∥∥​≤∥x∥

    for all 0≤θ≤10 \leq \theta \leq 10≤θ≤1.
Solution.

  1. Since AAA and BBB are positive definite, we can express A=SADASA−1A = S_AD_AS_A^{-1}A=SA​DA​SA−1​ and B=SBDBSB−1B = S_BD_BS_B^{-1}B=SB​DB​SB−1​, where DAD_ADA​ and DBD_BDB​ are diagonal matrices with positive entries on the diagonal. Thus, we may define Az=DAzA^z = D_A^zAz=DAz​ and Bz=DBzB^z = D_B^zBz=DBz​, where we raise each entry in the diagonal matrix to the zzz-th power. Thus, when expanding out ⟨y,BzA−zx⟩\inner{y, B^zA^{-z}x}⟨y,BzA−zx⟩, we get a finite linear combination of functions of the form aza^zaz with a>0a > 0a>0, hence entire.

  2. We wish to apply Hadamard's three-lines theorem. If Re⁡z=0\Re{z} = 0Rez=0, then observe that B0B^0B0 and A0A^0A0 have eigenvalues 111, so they have operator norm 111. Thus, by Cauchy-Schwarz,

    ∣⟨y,BzA−zx⟩∣≤∥x∥∥y∥.\abs{\inner{y, B^zA^{-z}x}} \leq \norm{x}\norm{y}.∣∣​⟨y,BzA−zx⟩∣∣​≤∥x∥∥y∥.

    If Re⁡z=1\Re{z} = 1Rez=1, write z=1+biz = 1 + biz=1+bi. The eigenvalues of A−ib,BibA^{-ib}, B^{ib}A−ib,Bib take the form λib=eiblog⁡λ\lambda^{ib} = e^{ib\log{\lambda}}λib=eiblogλ, which have absolute value 111. Hence,

    ∥BzA−z∥=∥BibBA−1A−bi∥≤∥Bib∥∥BA−1∥∥A−bi∥≤1\norm{B^zA^{-z}} = \norm{B^{ib}BA^{-1}A^{-bi}} \leq \norm{B^{ib}}\norm{BA^{-1}}\norm{A^{-bi}} \leq 1∥∥​BzA−z∥∥​=∥∥​BibBA−1A−bi∥∥​≤∥∥​Bib∥∥​∥∥​BA−1∥∥​∥∥​A−bi∥∥​≤1

    by assumption, so by Cauchy-Schwarz again,

    ∣⟨y,BzA−zx⟩∣≤∥BzA−zx∥∥y∥≤∥x∥∥y∥.\abs{\inner{y, B^zA^{-z}x}} \leq \norm{B^zA^{-z}x}\norm{y} \leq \norm{x}\norm{y}.∣∣​⟨y,BzA−zx⟩∣∣​≤∥∥​BzA−zx∥∥​∥y∥≤∥x∥∥y∥.

    Since fx,y(z)=⟨y,BzA−zx⟩f_{x,y}\p{z} = \inner{y, B^zA^{-z}x}fx,y​(z)=⟨y,BzA−zx⟩ is entire, it is in particular holomorphic near the strip S={z∣0≤Re⁡z≤1}S = \set{z \mid 0 \leq \Re{z} \leq 1}S={z∣0≤Rez≤1}, so by the Hadamard three-lines theorem, if we set M(t)=sup⁡b ∣fx,y(t+ib)∣M\p{t} = \sup_{b}\,\abs{f_{x,y}\p{t + ib}}M(t)=supb​∣fx,y​(t+ib)∣ and θ=1−t\theta = 1 - tθ=1−t for θ∈[0,1]\theta \in \br{0, 1}θ∈[0,1], we obtain

    M(θ)≤M(0)tM(1)1−t≤∥x∥∥y∥.M\p{\theta} \leq M\p{0}^t M\p{1}^{1-t} \leq \norm{x}\norm{y}.M(θ)≤M(0)tM(1)1−t≤∥x∥∥y∥.

    Hence, in particular, ∣⟨y,BθA−θx⟩∣≤∥x∥∥y∥\abs{\inner{y, B^\theta A^{-\theta}x}} \leq \norm{x}\norm{y}∣∣​⟨y,BθA−θx⟩∣∣​≤∥x∥∥y∥. Taking the supremum over ∥y∥=1\norm{y} = 1∥y∥=1, we obtain

    ∥BθA−θx∥≤∥x∥,\norm{B^\theta A^{-\theta}x} \leq \norm{x},∥∥​BθA−θx∥∥​≤∥x∥,

    which completes the proof.
