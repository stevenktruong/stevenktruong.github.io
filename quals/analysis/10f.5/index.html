Fall 2010 - Problem 5

density argument

Let R/Z\R/\ZR/Z denote the torus (whose elements we will write as cosets) and fix an irrational number α>0\alpha > 0α>0.

  1. Show that

    lim⁡N→∞1N∑n=0N−1f(nα+Z)=∫01f(x+Z) dx\lim_{N\to\infty} \frac{1}{N} \sum_{n=0}^{N-1} f\p{n\alpha + \Z} = \int_0^1 f\p{x + \Z} \,\diff{x}N→∞lim​N1​n=0∑N−1​f(nα+Z)=∫01​f(x+Z)dx

    for all continuous functions f ⁣:R/Z→R\func{f}{\R/\Z}{\R}f:R/Z→R.

    Hint: Consider first f(x)=e2πikxf\p{x} = e^{2\pi ikx}f(x)=e2πikx.

  2. Show that the conclusion is also true when fff is the characteristic function of a closed interval.
Solution.

  1. For notation, define

    EN(f)=1N∑n=0N−1f(nα+Z).E_N\p{f} = \frac{1}{N} \sum_{n=0}^{N-1} f\p{n\alpha + \Z}.EN​(f)=N1​n=0∑N−1​f(nα+Z).

    We follow the hint, and we first establish the conclusion for f(x)=e2πikxf\p{x} = e^{2\pi ikx}f(x)=e2πikx:

    1N∑n=0N−1e2πiknα=1N∑n=0N−1(e2πikα)n=1Ne2πikNα−1e2πikα−1,\begin{aligned} \frac{1}{N} \sum_{n=0}^{N-1} e^{2\pi ikn\alpha} &= \frac{1}{N} \sum_{n=0}^{N-1} \p{e^{2\pi ik\alpha}}^n \\ &= \frac{1}{N} \frac{e^{2\pi ikN\alpha} - 1}{e^{2\pi ik\alpha} - 1}, \end{aligned}N1​n=0∑N−1​e2πiknα​=N1​n=0∑N−1​(e2πikα)n=N1​e2πikα−1e2πikNα−1​,​

    which tends to 000 as N→∞N \to \inftyN→∞ for any k∈Zk \in \Zk∈Z. Indeed, α\alphaα is irrational, so the denominator is never 000. On the other hand,

    ∫01e2πikx dx=0\int_0^1 e^{2\pi ikx} \,\diff{x} = 0∫01​e2πikxdx=0

    since e2πikxe^{2\pi ikx}e2πikx is 111-periodic. Thus, lim⁡N→∞EN(e2πikx)=∫01e2πikx dx\lim_{N\to\infty} E_N\p{e^{2\pi ikx}} = \int_0^1 e^{2\pi ikx} \,\diff{x}limN→∞​EN​(e2πikx)=∫01​e2πikxdx. By linearity of integration, the equality is established for any trigonometric polynomial, i.e., any real linear combination of {e2πikx∣k∈Z}\set{e^{2\pi ikx} \mid k \in \Z}{e2πikx∣k∈Z}. Let AAA denote the set of trigonometric polynomials.

    Notice that by definition, AAA is a vector space over R\RR, AAA is closed under multiplication since e2πikxe2πiℓx=e2πi(k+ℓ)x∈Ae^{2\pi ikx} e^{2\pi i\ell x} = e^{2\pi i\p{k+\ell}x} \in Ae2πikxe2πiℓx=e2πi(k+ℓ)x∈A, and 1∈A1 \in A1∈A. Hence, AAA is a subalgebra of C(R/Z)C\p{\R/\Z}C(R/Z) which separates points, so by Stone-Weierstrass, AAA is dense in C(R/Z)C\p{\R/\Z}C(R/Z) with respect to the uniform norm.

    Let f∈C(R/Z)f \in C\p{\R/\Z}f∈C(R/Z) and ε>0\epsilon > 0ε>0. By density, we may pick g∈Ag \in Ag∈A such that ∥f−g∥L∞<ε\norm{f - g}_{L^\infty} < \epsilon∥f−g∥L∞​<ε. By our preliminary result, for NNN large enough, we also have ∣EN(g)−∫01g dx∣<ε\abs{E_N\p{g} - \int_0^1 g \,\diff{x}} < \epsilon∣∣​EN​(g)−∫01​gdx∣∣​<ε, so

    ∣EN(f)−∫01f(x+Z) dx∣≤∣EN(f)−EN(g)∣+∣EN(g)−∫01g(x+Z) dx∣+∣∫01g(x+Z) dx−∫01f(x+Z) dx∣≤1N∑n=0N−1∥f−g∥L∞+ε+∫01∥f−g∥L∞ dx=2∥f−g∥L∞+ε≤3ε.\begin{aligned} \abs{E_N\p{f} - \int_0^1 f\p{x + \Z} \,\diff{x}} &\leq \abs{E_N\p{f} - E_N\p{g}} + \abs{E_N\p{g} - \int_0^1 g\p{x + \Z} \,\diff{x}} + \abs{\int_0^1 g\p{x + \Z} \,\diff{x} - \int_0^1 f\p{x + \Z} \,\diff{x}} \\ &\leq \frac{1}{N} \sum_{n=0}^{N-1} \norm{f - g}_{L^\infty} + \epsilon + \int_0^1 \norm{f - g}_{L^\infty} \,\diff{x} \\ &= 2\norm{f - g}_{L^\infty} + \epsilon \\ &\leq 3\epsilon. \end{aligned}∣∣​EN​(f)−∫01​f(x+Z)dx∣∣​​≤∣EN​(f)−EN​(g)∣+∣∣​EN​(g)−∫01​g(x+Z)dx∣∣​+∣∣​∫01​g(x+Z)dx−∫01​f(x+Z)dx∣∣​≤N1​n=0∑N−1​∥f−g∥L∞​+ε+∫01​∥f−g∥L∞​dx=2∥f−g∥L∞​+ε≤3ε.​

    Sending ε→0\epsilon \to 0ε→0, we see

    lim⁡N→∞∑n=0N−1f(nα+Z)=∫01f(x+Z) dx,\lim_{N\to\infty} \sum_{n=0}^{N-1} f\p{n\alpha + \Z} = \int_0^1 f\p{x + \Z} \,\diff{x},N→∞lim​n=0∑N−1​f(nα+Z)=∫01​f(x+Z)dx,

    which completes the proof.

  2. Let f=χ[a,b]f = \chi_{\br{a,b}}f=χ[a,b]​, and pick sequences of continuous functions gk,hkg_k, h_kgk​,hk​ which converge to fff almost everywhere and such that 0≤gk≤f≤hk≤10 \leq g_k \leq f \leq h_k \leq 10≤gk​≤f≤hk​≤1. For example, we can pick gkg_kgk​ to be

    • 000 outside of [a,b]\br{a, b}[a,b] (possibly nowhere),
    • linear on [a,a+1k]\br{a, a + \frac{1}{k}}[a,a+k1​],
    • 111 on [a+1k,b−1k]\br{a + \frac{1}{k}, b - \frac{1}{k}}[a+k1​,b−k1​],
    • linear on [b−1k,b]\br{b - \frac{1}{k}, b}[b−k1​,b].

    Similarly, hkh_khk​ can be

    • 000 outside of [a−1k,b+1k]\br{a - \frac{1}{k}, b + \frac{1}{k}}[a−k1​,b+k1​] (possibly nowhere),
    • linear on [a−1k,a]\br{a - \frac{1}{k}, a}[a−k1​,a],
    • 111 on [a,b]\br{a, b}[a,b],
    • linear on [b,b+1k]\br{b, b + \frac{1}{k}}[b,b+k1​].

    By monotonicity and (1) applied to gk,hkg_k, h_kgk​,hk​, we have

    EN(gk)≤EN(f)≤EN(hk)  ⟹  ∫01gk(x+Z) dx≤lim inf⁡N→∞EN(f)≤lim sup⁡N→∞EN(f)≤∫01hk(x+Z) dx.\begin{gathered} E_N\p{g_k} \leq E_N\p{f} \leq E_N\p{h_k} \\ \implies \int_0^1 g_k\p{x + \Z} \,\diff{x} \leq \liminf_{N\to\infty} E_N\p{f} \leq \limsup_{N\to\infty} E_N\p{f} \leq \int_0^1 h_k\p{x + \Z} \,\diff{x}. \end{gathered}EN​(gk​)≤EN​(f)≤EN​(hk​)⟹∫01​gk​(x+Z)dx≤N→∞liminf​EN​(f)≤N→∞limsup​EN​(f)≤∫01​hk​(x+Z)dx.​

    Moreover, gkg_kgk​ and hkh_khk​ are dominated by 1∈L1(R/Z)1 \in L^1\p{\R/\Z}1∈L1(R/Z) and converge to fff almost everywhere, so by dominated convergence and the above inequality,

    ∫01f(x+Z) dx≤lim inf⁡N→∞EN(f)≤lim sup⁡N→∞EN(f)≤∫01f(x+Z) dx.\int_0^1 f\p{x + \Z} \,\diff{x} \leq \liminf_{N\to\infty} E_N\p{f} \leq \limsup_{N\to\infty} E_N\p{f} \leq \int_0^1 f\p{x + \Z} \,\diff{x}.∫01​f(x+Z)dx≤N→∞liminf​EN​(f)≤N→∞limsup​EN​(f)≤∫01​f(x+Z)dx.

    Thus, lim⁡N→∞EN(f)=∫01f(x+Z) dx\lim_{N\to\infty} E_N\p{f} = \int_0^1 f\p{x + \Z} \,\diff{x}limN→∞​EN​(f)=∫01​f(x+Z)dx, as required.
