Directional Derivatives

In 1D, if you had a function f(x)f\p{x}f(x), then the derivative dfdx\deriv{f}{x}dxdf​ tells you how much fff (the "height") changes when you move a little bit in the positive xxx direction.

With multiple variables, though, you have a lot more directions to work with, but the interpretation is the same:

Du⃗f(a,b)=∇f(a,b)⋅u⃗∥u⃗∥D_{\vec{u}}f\p{a, b} = \nabla f\p{a, b} \cdot \frac{\vec{u}}{\norm{\vec{u}}}Du​f(a,b)=∇f(a,b)⋅∥u∥u​

Du⃗f(a,b)D_{\vec{u}}f\p{a, b}Du​f(a,b) tells you how much fff (the "height") changes when you move a little bit in the direction specified by the vector u⃗\vec{u}u. Since this is a dot product, you already have another, more geometric formula for the directional derivative:

Du⃗f(a,b)=∥∇f(a,b)∥cos⁡θ.D_{\vec{u}}f\p{a, b} = \norm{\nabla f\p{a, b}} \cos{\theta}.Du​f(a,b)=∥∇f(a,b)∥cosθ.

θ\thetaθ is the angle between ∇f(a,b)\nabla f\p{a, b}∇f(a,b) and the vector u⃗\vec{u}u. Recall that cos⁡θ\cos{\theta}cosθ is between −1-1−1 and 111, where cos⁡0=1\cos{0} = 1cos0=1 and cos⁡π=−1\cos{\pi} = -1cosπ=−1. So, you immediately get two interesting interpretations of the gradient:

  1. The maximum rate of increase is in the direction ∇f(a,b)\nabla f\p{a, b}∇f(a,b).
  2. The maximum rate of decrease is in the direction −∇f(a,b)-\nabla f\p{a, b}−∇f(a,b).

Another interesting case is when cos⁡θ=0\cos{\theta} = 0cosθ=0, where θ=π2\theta = \frac{\pi}{2}θ=2π​, i.e., when ∇f(a,b)\nabla f\p{a, b}∇f(a,b) is orthogonal to u⃗\vec{u}u. In this case, Du⃗f(a,b)=0D_{\vec{u}}f\p{a, b} = 0Du​f(a,b)=0, i.e., if you move a little bit in a direction orthogonal to ∇f(a,b)\nabla f\p{a, b}∇f(a,b), then fff (essentially) stays constant. However, if you stay at the same height, that means you're moving along a level curve. This tells you another interpretation of the gradient:

The gradient of fff is orthogonal to the level curves of fff.

Using the interpretation that the gradient is orthogonal to level curves, you'll be able to find normal vectors for tangent planes much more easily.

Example 1.

If a surface is defined by F(x,y,z)=cF\p{x, y, z} = cF(x,y,z)=c, i.e., is a level curve of the function FFF, then you already know that ∇F\nabla F∇F will be orthogonal to this surface. Thus, you can use

n⃗=∇F\vec{n} = \nabla Fn=∇F

as your normal vector.
Example 2.

As a special case, if you have a surface defined by z=f(x,y)z = f\p{x, y}z=f(x,y), then you can view it as a level curve:

F(x,y,z)=f(x,y)−z=0F\p{x, y, z} = f\p{x, y} - z = 0F(x,y,z)=f(x,y)−z=0

Then as in the previous example, you can use

n⃗=∇F=⟨∂f∂x,∂f∂y,−1⟩,\vec{n} = \nabla F = \ang{\pderiv{f}{x}, \pderiv{f}{y}, -1},n=∇F=⟨∂x∂f​,∂y∂f​,−1⟩,

which coincides with the normal vector we've been using for surfaces.
Example 3.

Find a normal vector to the surface x2+y2+z2=1x^2 + y^2 + z^2 = 1x2+y2+z2=1 at the point (a,b,c)\p{a, b, c}(a,b,c).
Solution.

In this problem, F(x,y,z)=x2+y2+z2F\p{x, y, z} = x^2 + y^2 + z^2F(x,y,z)=x2+y2+z2 and the surface is a level surface of FFF. Thus, we can use

n⃗=∇F(a,b,c)=⟨2a,2b,2c⟩.\vec{n} = \nabla F\p{a, b, c} = \boxed{\ang{2a, 2b, 2c}}.n=∇F(a,b,c)=⟨2a,2b,2c⟩​.
