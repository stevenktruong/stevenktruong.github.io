Problem 3

Let VVV be an inner product space, let WWW be a finite-dimensional subspace of VVV. Prove that if x∉Wx \notin Wx∈/W, then there exists y∈W⊥y \in W^\perpy∈W⊥ with ⟨x,y⟩≠0\inner{x, y} \neq 0⟨x,y⟩=0.

Solution.

Recall that V=W⊕W⊥V = W \oplus W^\perpV=W⊕W⊥ (via orthogonal projection), so we can write

x=xW+xW⊥,x = x_W + x_{W^\perp},x=xW​+xW⊥​,

where xW∈Wx_W \in WxW​∈W and xW⊥∈W⊥x_{W^\perp} \in W^\perpxW⊥​∈W⊥. Note that xW⊥≠0x_{W^\perp} \neq 0xW⊥​=0: if this were not the case, then x=xW∈Wx = x_W \in Wx=xW​∈W, which is impossible. Thus, if we set y=xW⊥y = x_{W^\perp}y=xW⊥​, then

⟨x,y⟩=⟨xW+y,y⟩=⟨xW,y⟩+⟨y,y⟩=∥y∥2>0.\begin{aligned} \inner{x, y} &= \inner{x_W + y, y} \\ &= \inner{x_W, y} + \inner{y, y} \\ &= \norm{y}^2 \\ &> 0. \end{aligned}⟨x,y⟩​=⟨xW​+y,y⟩=⟨xW​,y⟩+⟨y,y⟩=∥y∥2>0.​

The penultimate inequality follows from the fact that yyy is orthogonal to every vector in WWW, and ∥y∥\norm{y}∥y∥ is strictly larger than 000 since y≠0y \neq 0y=0.

Problem 4

Let VVV be a finite-dimensional inner product space, let WWW be a subspace of VVV. Prove that V/WV/WV/W is isomorphic to W⊥W^\perpW⊥.

Solution.

There are two different ways to do this:

  1. Every finite-dimensional vector space over a field F\FF is isomorphic to Fn\F^nFn for some nnn, so we can try proving dim⁡(V/W)=dim⁡(W⊥)\dim\p{V/W} = \dim\p{W^\perp}dim(V/W)=dim(W⊥).
  2. We can try to find a linear isomorphism T ⁣:W⊥→V/WT\colon W^\perp \to V/WT:W⊥→V/W.

Method 1

Let dim⁡V=n\dim V = ndimV=n and dim⁡W=m\dim W = mdimW=m. Recall that because V,WV, WV,W are both finite-dimensional that dim⁡(V/W)=n−m\dim\p{V/W} = n - mdim(V/W)=n−m. To show dim⁡(W⊥)=n−m\dim\p{W^\perp} = n - mdim(W⊥)=n−m, we need to find a basis for W⊥W^\perpW⊥.

Let {v1,…,vm}\set{v_1, \ldots, v_m}{v1​,…,vm​} be an orthonormal basis (it's very important that we pick an orthonormal basis for this to work) for WWW, and extend it into an orthonormal basis β={v1,…,vm,…,vn}\beta = \set{v_1, \ldots, v_m, \ldots, v_n}β={v1​,…,vm​,…,vn​} for VVV. We will show that {vm+1,…,vn}\set{v_{m+1}, \ldots, v_n}{vm+1​,…,vn​} is a basis for W⊥W^\perpW⊥.

First, the set {vm+1,…,vn}\set{v_{m+1}, \ldots, v_n}{vm+1​,…,vn​} is linearly independent since β\betaβ was. Next, we need to show that it spans W⊥W^\perpW⊥. Let v∈W⊥v \in W^\perpv∈W⊥. Since β\betaβ was an orthonormal basis for VVV, we can write

v=⟨v,v1⟩v1+⋯+⟨v,vm⟩vm+⋯+⟨v,vn⟩vn.v = \inner{v, v_1}v_1 + \cdots + \inner{v, v_m}v_m + \cdots + \inner{v, v_n}v_n.v=⟨v,v1​⟩v1​+⋯+⟨v,vm​⟩vm​+⋯+⟨v,vn​⟩vn​.

But v∈W⊥v \in W^\perpv∈W⊥ and v1,…,vm∈W⊥v_1, \ldots, v_m \in W^\perpv1​,…,vm​∈W⊥, so ⟨v,v1⟩=⋯=⟨v,vm⟩=0\inner{v, v_1} = \cdots = \inner{v, v_m} = 0⟨v,v1​⟩=⋯=⟨v,vm​⟩=0. Thus,

v=⟨v,vm+1⟩vm+1+⋯+⟨v,vn⟩vn.v = \inner{v, v_{m+1}}v_{m+1} + \cdots + \inner{v, v_n}v_n.v=⟨v,vm+1​⟩vm+1​+⋯+⟨v,vn​⟩vn​.

This means v∈span⁡{vm+1,…,vn}v \in \span\set{v_{m+1}, \ldots, v_n}v∈span{vm+1​,…,vn​}, so the set spans. Thus, this set is a basis for W⊥W^\perpW⊥ and has n−mn - mn−m vectors, so dim⁡W⊥=n−m\dim W^\perp = n - mdimW⊥=n−m.

Method 2

Define T ⁣:W⊥→V/WT\colon W^\perp \to V/WT:W⊥→V/W by

T(v)=v+W.T\p{v} = v + W.T(v)=v+W.

This map is linear: given u,v∈W⊥u, v \in W^\perpu,v∈W⊥ and a scalar c∈Fc \in \Fc∈F, the definitions of addition and scalar multiplication in V/WV/WV/W tell us

T(cu+v)=(cu+v)+W=c(u+W)+(v+W)=cT(u)+T(v),\begin{aligned} T\p{cu + v} &= \p{cu + v} + W \\ &= c\p{u + W} + \p{v + W} \\ &= cT\p{u} + T\p{v}, \end{aligned}T(cu+v)​=(cu+v)+W=c(u+W)+(v+W)=cT(u)+T(v),​

so TTT is linear. Let's show that TTT is one-to-one: let v∈N(T)v \in N\p{T}v∈N(T). Then

v+W=T(v)=W,v + W = T\p{v} = W,v+W=T(v)=W,

so v∈Wv \in Wv∈W. But v∈W⊥v \in W^\perpv∈W⊥, so v∈W∩W⊥={0⃗}v \in W \cap W^\perp = \set{\vec{0}}v∈W∩W⊥={0}, and so v=0⃗v = \vec{0}v=0. This shows that N(T)={0}N\p{T} = \set{0}N(T)={0}, so TTT is one-to-one.

Now let's show that TTT is onto. Let v+W∈V/Wv + W \in V/Wv+W∈V/W. Note that we can't just say that

T(v)=v+W.T\p{v} = v + W.T(v)=v+W.

This is because vvv does not have to be in W⊥W^\perpW⊥. However, if we could replace vvv with a vector y∈W⊥y \in W^\perpy∈W⊥ such that v+W=y+Wv + W = y + Wv+W=y+W, then T(y)=v+WT\p{y} = v + WT(y)=v+W and we are done. The "natural" way to do this is to write

v=vW+vW⊥v = v_W + v_{W^\perp}v=vW​+vW⊥​

where vW∈Wv_W \in WvW​∈W and vW⊥∈W⊥v_{W^\perp} \in W^\perpvW⊥​∈W⊥ as in Problem 3. Thus, by definition of v+Wv + Wv+W, we have

v+W=(vW+vW⊥)+W=vW⊥+W,v + W = \p{v_W + v_{W^\perp}} + W = v_{W^\perp} + W,v+W=(vW​+vW⊥​)+W=vW⊥​+W,

since vW∈Wv_W \in WvW​∈W. Hence, we have

T(vW⊥)=vW⊥+W=v+W,T\p{v_{W^\perp}} = v_{W^\perp} + W = v + W,T(vW⊥​)=vW⊥​+W=v+W,

so TTT is onto.

We have shown that TTT is an invertible linear map, so it's automatically an isomorphism.
