Formal Calculus

"Formal" here means that you're not actually doing calculus (i.e., you're not taking any limits), but you're pretending that you can. This is what's happening in the following problem:

Example 1.

Let T ⁣:F[x]→F[x]\func{T}{\mathbb{F}\br{x}}{\mathbb{F}\br{x}}T:F[x]→F[x] be defined by

T(f)=∫0xf(t) dt.T\p{f} = \int_0^x f\p{t} \,\diff{t}.T(f)=∫0x​f(t)dt.

Show that TTT is linear.

Even though we're using the integral symbol, this does not mean that we're using the Riemann integral to define TTT. Remember that the Riemann integral is defined by taking the limit of a Riemann sum of a function f ⁣:[a,b]→R\func{f}{\br{a,b}}{\R}f:[a,b]→R. To differentiate between these two, I will use TRiemannT_{\mathrm{Riemann}}TRiemann​ to denote the Riemann integral. In this instance, TRiemann ⁣:{polynomial functions}→{polynomial functions}\func{T_{\mathrm{Riemann}}}{\set{\text{polynomial functions}}}{\set{\text{polynomial functions}}}TRiemann​:{polynomial functions}→{polynomial functions}, with

TRiemann(f)=∫0xf(t) dt=lim⁡∣P∣→0∑ti∈Pf(ti)(ti+1−ti).T_{\mathrm{Riemann}}\p{f} = \int_0^x f\p{t} \,\diff{t} = \lim_{\abs{\mathcal{P}}\to0} \sum_{t_i \in \mathcal{P}} f(t_i)\p{t_{i+1} - t_i}.TRiemann​(f)=∫0x​f(t)dt=∣P∣→0lim​ti​∈P∑​f(ti​)(ti+1​−ti​).

(You don't need to pay that much attention to the limit part. It's just there to remind you of how little TTT uses when compared to TRiemannT_{\mathrm{Riemann}}TRiemann​.)

TTT and TRiemannT_{\mathrm{Riemann}}TRiemann​ may look the same, but they are actually very different. TTT is a map between things that we call formal polynomials, which essentially means that we don't think of xxx as a variable, but as just a (mostly meaningless) symbol. On the other hand, TRiemannT_{\mathrm{Riemann}}TRiemann​ is defined between polynomial functions, which means that we think of xxx as a variable that we can plug things into.

The most common issue I saw was that students confused TTT with TRiemannT_{\mathrm{Riemann}}TRiemann​, which is understandable, but still a big mistake in this class. It's true that TRiemannT_{\mathrm{Riemann}}TRiemann​ is linear, but it doesn't tell you anything about linearity of TTT because TTT is defined formally instead of with Riemann sums. Like I said at the beginning, "formally" means you're only pretending you can do calculus. If you could do calculus on F[x]\mathbb{F}\br{x}F[x] (which you can never actually do), then given a polynomial f=∑i=0naixif = \sum_{i=0}^n a_i x^if=∑i=0n​ai​xi, the only reasonable meaning of the integral is

∫0xf(t) dt=∑i=0naii+1ti+1∣t=0t=x=∑i=0naii+1xi+1.\int_0^x f\p{t} \,\diff{t} = \left. \sum_{i=0}^n \frac{a_i}{i+1} t^{i+1} \right\rvert_{t=0}^{t=x} = \sum_{i=0}^n \frac{a_i}{i+1} x^{i+1}.∫0x​f(t)dt=i=0∑n​i+1ai​​ti+1∣∣​t=0t=x​=i=0∑n​i+1ai​​xi+1.

So, the actual definition of TTT (and how you're supposed to interpret the problem) is

T(∑i=0naixi)=∑i=0naii+1xi+1,T\p{\sum_{i=0}^n a_ix^i} = \sum_{i=0}^n \frac{a_i}{i+1} x^{i+1},T(i=0∑n​ai​xi)=i=0∑n​i+1ai​​xi+1,

i.e., it's what you get if you could apply the fundamental theorem of calculus. But we can't actually use it, which is why TTT is called formal integration.

Correctly interpreting problems is a skill you have to learn in math, so it's important to take a step back and think about whether what you write down makes sense. For example, you could ask yourself:

If F=Z2\mathbb{F} = \Z_2F=Z2​, then does it make sense to use the fundamental theorem of calculus?

Hopefully, you would answer no, because calculus can only be done if you can take limits, which you can't do in Z2\Z_2Z2​ (or most fields, for that matter). This means that the only reasonable interpretation of the integral in TTT is to compute it formally.

Exercise 1.
