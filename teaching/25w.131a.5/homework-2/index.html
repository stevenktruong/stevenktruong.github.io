  1. First, notice that because ∣b∣≥0\abs{b} \geq 0∣b∣≥0, we also have a≥0a \geq 0a≥0. There are two cases: b≥0b \geq 0b≥0 and b<0b < 0b<0. If b≥0b \geq 0b≥0, then

    −a≤0≤b=∣b∣≤a.-a \leq 0 \leq b = \abs{b} \leq a.−a≤0≤b=∣b∣≤a.

    Similarly, if b<0b < 0b<0, then

    −a≤−∣b∣=b≤0≤a.-a \leq -\abs{b} = b \leq 0 \leq a.−a≤−∣b∣=b≤0≤a.

    ∣a∣=∣(a−b)+b∣≤∣a−b∣+∣b∣  ⟹  ∣a∣−∣b∣≤∣a−b∣.\abs{a} = \abs{\p{a - b} + b} \leq \abs{a - b} + \abs{b} \implies \abs{a} - \abs{b} \leq \abs{a - b}.∣a∣=∣(a−b)+b∣≤∣a−b∣+∣b∣⟹∣a∣−∣b∣≤∣a−b∣.

    This is true for any a,b∈Ra, b \in \Ra,b∈R. In particular, swapping aaa and bbb gives the inequality

    ∣b∣−∣a∣≤∣b−a∣=∣a−b∣.  ⟹  −∣a−b∣≤∣a∣−∣b∣.\abs{b} - \abs{a} \leq \abs{b - a} = \abs{a - b}. \implies -\abs{a - b} \leq \abs{a} - \abs{b}.∣b∣−∣a∣≤∣b−a∣=∣a−b∣.⟹−∣a−b∣≤∣a∣−∣b∣.

Common Mistakes

The main issue I saw with proofs was that students just said to "repeat this process" infinitely many times, but "this process" was not very clearly defined. For example, I saw this multiple times:

By density of Q\QQ, there exists r1∈Qr_1 \in \Qr1​∈Q such that a<r1<ba < r_1 < ba<r1​<b. By density again, there exists r2∈Qr_2 \in \Qr2​∈Q such that r1<r2<br_1 < r_2 < br1​<r2​<b. Repeating this process, we get infinitely many rational numbers between aaa and bbb.

While the idea is correct, the details are not all there. For example, how should I pick the endpoints for the next step? If I used r1<br_1 < br1​<b again, then I would get r3∈Qr_3 \in \Qr3​∈Q such that r1<r3<br_1 < r_3 < br1​<r3​<b. However, maybe I'm unlucky and it turns out that r3=r2r_3 = r_2r3​=r2​. You have to describe a process that lets me always pick a brand new rational number.
