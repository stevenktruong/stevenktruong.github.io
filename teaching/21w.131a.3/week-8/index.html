I emphasized the word "carefully" because it's the reason why I was fairly strict with grading. Many of you had something like this as the first line:

If {sn}n\set{s_n}_n{sn​}n​ converges to sss and {tn}n\set{t_n}_n{tn​}n​ converges to ttt with t≠0t \neq 0t=0, then {tnsn}n\set{\frac{t_n}{s_n}}_n{sn​tn​​}n​ converges to ts\frac{t}{s}st​.

Remember that you need to check assumptions before using a fact. The assumptions for Theorem 9.6 are that lim⁡n→∞sn\displaystyle\lim_{n\to\infty} s_nn→∞lim​sn​ and lim⁡n→∞tn\displaystyle\lim_{n\to\infty} t_nn→∞lim​tn​ exist. In other words, if you wrote the line above before checking that the limits of the numerator and denominator exist, you used Theorem 9.6 before checking all assumptions. Introduction

Definition

Suppose f ⁣:D→R\func{f}{D}{\R}f:D→R is a function. Then fff is continuous at x0∈Dx_0 \in Dx0​∈D if one of the following holds:

  1. If {xn}n\set{x_n}_n{xn​}n​ is a sequence in DDD converging to x0x_0x0​, then {f(xn)}n\set{f\p{x_n}}_n{f(xn​)}n​ converges to f(x0)f\p{x_0}f(x0​).
  2. For any ε>0\epsilon > 0ε>0, there exists δ>0\delta > 0δ>0 such that whenever y∈Dy \in Dy∈D satisfies ∣x0−y∣<δ\abs{x_0 - y} < \delta∣x0​−y∣<δ, then ∣f(x0)−f(y)∣<ε\abs{f\p{x_0} - f\p{y}} < \epsilon∣f(x0​)−f(y)∣<ε.

From lecture, you already know that (i) and (ii) are equivalent definitions, which means we can use whichever one's more convenient for us. As a rule of thumb, (i) is easier to use to prove something is discontinuous, and (ii) is easier to use to prove something is continuous.

Another consequence of the sequence definition of continuity is that facts about sequences translate directly into facts about continuity.

Theorem

Suppose f ⁣:D→R\func{f}{D}{\R}f:D→R and g ⁣:D→R\func{g}{D}{\R}g:D→R are continuous functions. Then:

  1. f+gf + gf+g is continuous.
  2. fgfgfg is continuous.
  3. fg\frac{f}{g}gf​ is continuous at all points x0∈Dx_0 \in Dx0​∈D such that g(x0)≠0g\p{x_0} \neq 0g(x0​)=0.
Proof.
(sketch)

Continuity at a point x0∈Dx_0 \in Dx0​∈D basically boils down to sequences. For f+gf + gf+g, if {xn}n\set{x_n}_n{xn​}n​ is a sequence in DDD that converges to x0x_0x0​, then we need to show that {(f+g)(xn)}n\set{\p{f + g}\p{x_n}}_n{(f+g)(xn​)}n​ converges to (f+g)(x0)\p{f + g}\p{x_0}(f+g)(x0​), but this boils down to the limit laws for sequences. □\square□

This theorem tells us that sequences help us with proving continuity. The converse is also true: we can use continuity to help us calculate limits.

To end this section, here's an important connection between density and continuity:

Proposition

Let f ⁣:R→R\func{f}{\R}{\R}f:R→R and g ⁣:R→R\func{g}{\R}{\R}g:R→R be continuous functions such that f(q)=g(q)f\p{q} = g\p{q}f(q)=g(q) for all q∈Qq \in \Qq∈Q. Then f=gf = gf=g everywhere.
Remark.

The idea behind this fact is that a continuous function will "fill in gaps" through limits. For example, let's say I give you all the information about a function f(x)f\p{x}f(x) except at a single point:

Because of continuity, there's only one way you can fill in the gap. In other words, information about a continuous function except at one point is the same thing as information about the continuous function everywhere.

The proposition says something a bit stronger: information about a continuous function at the rational numbers is the same as information about the function on the whole real line.
Proof.

To prove the statement, we just need to use density of Q\QQ. Let x0∈Rx_0 \in \Rx0​∈R, so by density of Q\QQ, there exists a sequence of rational numbers {qn}n\set{q_n}_n{qn​}n​ which converge to x0x_0x0​. Since we know that fff and ggg agree at every rational, we have f(qn)=g(qn)f\p{q_n} = g\p{q_n}f(qn​)=g(qn​) for all nnn. Then by the sequence definition of continuity,

f(x0)=lim⁡n→∞f(qn)=lim⁡n→∞g(qn)=g(x0).f\p{x_0} = \lim_{n\to\infty} f\p{q_n} = \lim_{n\to\infty} g\p{q_n} = g\p{x_0}.f(x0​)=n→∞lim​f(qn​)=n→∞lim​g(qn​)=g(x0​).

Since x0x_0x0​ was arbitrary, this means f=gf = gf=g everywhere on R\RR, which was what we wanted to show. □\square□
