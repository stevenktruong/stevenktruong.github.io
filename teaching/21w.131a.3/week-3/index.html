Induction

Example 1.

Prove 13+23+⋯+n3=(1+2+⋯+n)21^3 + 2^3 + \cdots + n^3 = \p{1 + 2 + \cdots + n}^213+23+⋯+n3=(1+2+⋯+n)2 for all positive integers nnn.
Solution.

Base case: (n=1n = 1n=1) Plugging n=1n = 1n=1 into the left side, we get

13=1=12,1^3 = 1 = 1^2,13=1=12,

Inductive step: Suppose the equation is true for n=kn = kn=k, i.e., assume that

(1+2+⋯+k+(k+1))2=((1+2+⋯+k)+(k+1))2=(1+2+⋯+k)2+2(k+1)(1+2+⋯+k)+(k+1)2=(13+23+⋯+k3)+2(k+1)⋅k(k+1)2+(k+1)2=(13+23+⋯+k3)+k(k+1)2+(k+1)2=(13+23+⋯+k3)+(k+1)2(k+1)=13+23+⋯+k3+(k+1)3,\begin{aligned} \p{1 + 2 + \cdots + k + \p{k + 1}}^2 &= \p{\p{1 + 2 + \cdots + k} + \p{k + 1}}^2 \\ &= \p{1 + 2 + \cdots + k}^2 + 2\p{k + 1}\p{1 + 2 + \cdots + k} + \p{k + 1}^2 \\ &= \p{1^3 + 2^3 + \cdots + k^3} + 2\p{k + 1} \cdot \frac{k\p{k + 1}}{2} + \p{k + 1}^2 \\ &= \p{1^3 + 2^3 + \cdots + k^3} + k\p{k + 1}^2 + \p{k + 1}^2 \\ &= \p{1^3 + 2^3 + \cdots + k^3} + \p{k + 1}^2\p{k + 1} \\ &= 1^3 + 2^3 + \cdots + k^3 + \p{k + 1}^3, \end{aligned}(1+2+⋯+k+(k+1))2​=((1+2+⋯+k)+(k+1))2=(1+2+⋯+k)2+2(k+1)(1+2+⋯+k)+(k+1)2=(13+23+⋯+k3)+2(k+1)⋅2k(k+1)​+(k+1)2=(13+23+⋯+k3)+k(k+1)2+(k+1)2=(13+23+⋯+k3)+(k+1)2(k+1)=13+23+⋯+k3+(k+1)3,​

Rational Numbers Q\QQ

Theorem (rational roots theorem)

Consider the equation

cnxn+cn−1xn−1+⋯+c0=0,c_n x^n + c_{n-1} x^{n-1} + \cdots + c_0 = 0,cn​xn+cn−1​xn−1+⋯+c0​=0,

where c0,c1,…,cn∈Zc_0, c_1, \ldots, c_n \in \Zc0​,c1​,…,cn​∈Z. Let r=cdr = \frac{c}{d}r=dc​ be a rational solution to the equation, where rrr is simplified so that ccc and ddd have no factors in common. Then ccc is a factor of c0c_0c0​ and ddd is a factor of cnc_ncn​.

The intuition for the proof is as follows: cd\frac{c}{d}dc​ is a root of dx−cdx - cdx−c, so we want to factor the polynomial into

cnxn+cn−1xn−1+⋯+c0=(dx−c)(an−1xn−1+⋯+a0)=dan−1xn+⋯−ca0.\begin{aligned} c_n x^n + c_{n-1} x^{n-1} + \cdots + c_0 &= \p{dx - c}\p{a_{n-1}x^{n-1} + \cdots + a_0} \\ &= da_{n-1} x^n + \cdots - ca_0. \end{aligned}cn​xn+cn−1​xn−1+⋯+c0​​=(dx−c)(an−1​xn−1+⋯+a0​)=dan−1​xn+⋯−ca0​.​

If we compare coefficients, we see cn=dan−1c_n = da_{n-1}cn​=dan−1​ and −ca0=c0-ca_0 = c_0−ca0​=c0​, so if an−1a_{n-1}an−1​ and a0a_0a0​ are integers, then ddd should divide cnc_ncn​ and ccc should divide c0c_0c0​. We can't guarantee that an−1a_{n-1}an−1​ and a0a_0a0​ are integers, though, so this isn't a proof.

Real Numbers R\RR

Definition

One problem with Q\QQ is that it has lots of gaps. For example, you can "separate" Q\QQ into

Q={q∈Q∣q2<2}∪{q∈Q∣q2>2},\Q = \set{q \in \Q \mid q^2 < 2} \cup \set{q \in \Q \mid q^2 > 2},Q={q∈Q∣q2<2}∪{q∈Q∣q2>2},

i.e., Q\QQ "jumps over" 2\sqrt{2}2​. This is why we want to work with the real numbers; it has no gaps.

Definition (the real numbers)

The real numbers, denoted R\RR, is the ordered field with the least upper bound property.

In other words, R\RR is what you "expect it to be":

  • Ordered means you can compare two numbers, i.e., they're equal or one is bigger than the other.
  • Field means you can add, subtract, multiply, and divide like you always did.

The least upper bound property takes a bit more work to understand.

The Least Upper Bound Property

Definition

Let S⊆RS \subseteq \RS⊆R. Then

  • SSS is bounded above if there exists B∈RB \in \RB∈R such that ∀x∈S\forall x \in S∀x∈S, x≤Bx \leq Bx≤B. We call BBB an upper bound for SSS.
  • SSS is bounded below if there exists L∈RL \in \RL∈R such that ∀x∈S\forall x \in S∀x∈S, L≤xL \leq xL≤x. We call LLL a lower bound for SSS.
  • SSS is bounded if it is both bounded above and bounded below.
Example 3.

If a,ba, ba,b are finite, then:

  • (−∞,b)\p{-\infty, b}(−∞,b) is bounded from above (by bbb) but not from below,
  • (a,∞)\p{a, \infty}(a,∞) is bounded from below (by aaa) but not from above,
  • (a,b)\p{a, b}(a,b) is bounded.

Now we can understand what the least upper bound property is:

Definition

R\RR has the least upper bound property, that is:

If S⊆RS \subseteq \RS⊆R is non-empty and bounded above, then there exists x∗∈Rx^* \in \Rx∗∈R such that:

  1. x∗x^*x∗ is an upper bound for SSS, and
  2. if BBB is another upper bound for SSS, then x∗≤Bx^* \leq Bx∗≤B.

When this happens, we call x∗x^*x∗ the supremum of SSS and write x∗=sup⁡Sx^* = \sup{S}x∗=supS.

The least upper bound is what it sounds like: it's an upper bound that's smaller than any other upper bound.

Proposition

If sup⁡S\sup{S}supS exists, then it is unique.
Proof.

Suppose x,x′x, x'x,x′ are two least upper bounds for SSS.

  • Since xxx is a least upper bound and x′x'x′ is another upper bound for SSS, we get x≤x′x \leq x'x≤x′.
  • Similarly, x′x'x′ is a least upper bound and xxx is an upper bound for SSS, so x′≤xx' \leq xx′≤x.

Putting these together, we get x=x′x = x'x=x′. □\square□

Definition

We say that R\RR has the greatest lower bound property if:

Whenever S⊆RS \subseteq \RS⊆R is non-empty and bounded below, there exists x∗∈Rx_* \in \Rx∗​∈R such that:

  1. x∗x_*x∗​ is a lower bound for SSS, and
  2. if LLL is another lower bound for SSS, then L≤x∗L \leq x_*L≤x∗​.

In this case, we say x∗x_*x∗​ is the infimum of SSS, and we write x∗=inf⁡Sx_* = \inf{S}x∗​=infS.
Proposition

R\RR has the greatest lower bound property.
Proof.

The closest thing we have to this is the least upper bound property, so we want to try to use it somehow.

The idea is try to reverse engineer what the greatest lower bound for SSS would be. If we "mirror" SSS, i.e., consider the set −S={−x∣x∈S}-S = \set{-x \mid x \in S}−S={−x∣x∈S}, then we get something like this:

Based on this picture, inf⁡S\inf{S}infS should correspond to sup⁡ (−S)\sup\,\p{-S}sup(−S) after the reflection. Since S≠∅S \neq \emptysetS=∅, this tells us that −S≠∅-S \neq \emptyset−S=∅ also.
