Monotone Sequences

Definition

Let {sn}n≥m\set{s_n}_{n \geq m}{sn​}n≥m​ be a sequence.

  1. {sn}n≥m\set{s_n}_{n \geq m}{sn​}n≥m​ is increasing if sn≤sn+1s_n \leq s_{n+1}sn​≤sn+1​ for all n≥mn \geq mn≥m.
  2. {sn}n≥m\set{s_n}_{n \geq m}{sn​}n≥m​ is decreasing if sn+1≤sns_{n+1} \leq s_nsn+1​≤sn​ for all n≥mn \geq mn≥m.
  3. {sn}n≥m\set{s_n}_{n \geq m}{sn​}n≥m​ is monotone if it's increasing or decreasing.

A general trend in math is that monotone things tend to have nice properties, which is why we're interested in them. With sequences, we have the following theorem:

Theorem

All bounded monotone sequences converge.

As you may guess, a bounded increasing sequence converges to its supremum, and a bounded decreasing sequence converges to its infimum. This is a very useful tool when working with recursive sequences.

You may be tempted to do the following: (which is wrong!)

If we take limits on both sides, then we get the equation s=s2s = s^2s=s2, so s=0s = 0s=0 or s=1s = 1s=1. Since sn<1s_n < 1sn​<1 for n≥1n \geq 1n≥1, this means s≠1s \neq 1s=1, so s=0s = 0s=0.

The problem is the very first step: this fake proof is trying to apply the limit laws without knowing if the limits exist or not. This is why it's important to do (1) before doing (2).

  1. Generally, when you're given a recursive sequence like this, you want to prove that it's bounded and monotone (usual by induction, though it's not always necessary). We want to figure out whether the sequence is increasing or decreasing, so a good way to start is to write out a few terms and take a guess:

    s1=1, s2=23, s3=13.s_1 = 1,\ s_2 = \frac{2}{3},\ s_3 = \frac{1}{3}.s1​=1, s2​=32​, s3​=31​.

Generally, however, it's usually much easier to show that sn+1−sn≤0s_{n+1} - s_n \leq 0sn+1​−sn​≤0, so let's see what happens:

    sn+1−sn=nn+1sn2−sn≤n+1n+1sn2−sn=sn(sn−1).\begin{aligned} s_{n+1} - s_n &= \frac{n}{n + 1} s_n^2 - s_n \\ &\leq \frac{n + 1}{n + 1} s_n^2 - s_n \\ &= s_n\p{s_n - 1}. \end{aligned}sn+1​−sn​​=n+1n​sn2​−sn​≤n+1n+1​sn2​−sn​=sn​(sn​−1).​

    Base case: (n=1n = 1n=1)

    s1=1≤1s_1 = 1 \leq 1s1​=1≤1, so the base case holds.

    Inductive step:

    sn+1=nn+1sn2≤n+1n+1⋅12=1.s_{n+1} = \frac{n}{n + 1} s_n^2 \leq \frac{n + 1}{n + 1} \cdot 1^2 = 1.sn+1​=n+1n​sn2​≤n+1n+1​⋅12=1.

Constructing Sequences

This is an important technique and is best illustrated through an example.

Example 2.

Let a∈Ra \in \Ra∈R. Then there exists a sequence of rational numbers {qn}n\set{q_n}_n{qn​}n​ which decrease to aaa.
Solution.

The main idea is to use density of rationals over and over again, but in a careful way to get the properties we want. There are two things we want from our sequence:

  1. It gets closer and closer to aaa. For concreteness, we want a≤qn≤a+1na \leq q_n \leq a + \frac{1}{n}a≤qn​≤a+n1​ for each n≥1n \geq 1n≥1, but you can replace 1n\frac{1}{n}n1​ with any other positive sequence that converges to 000 (another good choice is 12n\frac{1}{2^n}2n1​).
  2. It is a decreasing sequence.

To construct a sequence, we generally want to do it by induction and making sure that at each step, we pick a rational number that does both of these.

Base case: (n=1n = 1n=1)

Since Q\QQ is dense in R\RR, there exists a rational q1∈(a,a+1)q_1 \in \p{a, a + 1}q1​∈(a,a+1). To illustrate the inductive step, I will also do the case n=2n = 2n=2 by hand:

Let ε=min⁡ {12,q1−a}\epsilon = \min\,\set{\frac{1}{2}, q_1 - a}ε=min{21​,q1​−a}, so (a,a+ε)\p{a, a + \epsilon}(a,a+ε) is a non-empty interval. Thus, because Q\QQ is dense, there exists a rational q2∈(a,a+ε)q_2 \in \p{a, a + \epsilon}q2​∈(a,a+ε), and this satisfies both (1) and (2).

Cauchy Sequences

Definition

A sequence {sn}n\set{s_n}_n{sn​}n​ is Cauchy if for all ε>0\epsilon > 0ε>0, there exists N∈RN \in \RN∈R such that ∣sn−sk∣<ε\abs{s_n - s_k} < \epsilon∣sn​−sk​∣<ε whenever n>Nn > Nn>N and k>Nk > Nk>N.

This definition looks a lot like the definition of a limit, but there is an important difference: there is no limit specified. Instead, we just put another term of the sequence in the absolute values.

You're probably thinking "what's the point?" which is a valid question. The point is the following theorem:

Theorem

A sequence is Cauchy if and only if it is convergent.

The "if and only if" tells you that the definition of Cauchy sequences and the definition of a convergent sequence describe the same thing. The key difference, though, is that in the definition of a Cauchy sequence, we don't have to know what the limit is. Generally, this is very useful when you do more abstract math since you have less information about the sequences you're working with.

Example 3.

Let's say we want to show that an infinite sum converges: ∑n=1∞sn\displaystyle \sum_{n=1}^\infty s_nn=1∑∞​sn​, which is defined to be lim⁡N→∞∑n=1Nsn\displaystyle \lim_{N\to\infty} \sum_{n=1}^N s_nN→∞lim​n=1∑N​sn​. We're already cheating here, since we're defining the infinite sum to be the limit of something that we don't even know exists or not.

We can't really use the definition of a limit since we don't know if the limit even exists, but if we use the Cauchy definition of a convergent sequence, then we're in good shape: we just need to show that the sequence {∑n=1Nsn}N\set{\displaystyle\sum_{n=1}^N s_n}_N{n=1∑N​sn​}N​ is Cauchy, and this is fine since every term in the sequence is a finite sum, so we're dealing with things that we know exist.
Definition

Let {sn}n\set{s_n}_n{sn​}n​ be a sequence.

  1. The limit superior of {sn}n\set{s_n}_n{sn​}n​ is

    lim sup⁡n→∞sn=lim⁡N→∞sup⁡ {sn∣n>N}.\limsup_{n\to\infty} s_n = \lim_{N\to\infty} \sup\,\set{s_n \mid n > N}.n→∞limsup​sn​=N→∞lim​sup{sn​∣n>N}.

  2. The limit inferior of {sn}n\set{s_n}_n{sn​}n​ is

    lim sup⁡n→∞sn=lim⁡N→∞inf⁡ {sn∣n>N}.\limsup_{n\to\infty} s_n = \lim_{N\to\infty} \inf\,\set{s_n \mid n > N}.n→∞limsup​sn​=N→∞lim​inf{sn​∣n>N}.
Remark.

Let uN=sup⁡ {sn∣n>N}u_N = \sup\,\set{s_n \mid n > N}uN​=sup{sn​∣n>N}. Notice that {sn∣n>N+1}⊆{sn∣n>N}\set{s_n \mid n > N + 1} \subseteq \set{s_n \mid n > N}{sn​∣n>N+1}⊆{sn​∣n>N}, so by the properties of the supremum,

sup⁡ {sn∣n>N+1}≤sup⁡ {sn∣n>N}  ⟹  uN+1≤uN,\sup\,\set{s_n \mid n > N + 1} \leq \sup\,\set{s_n \mid n > N} \implies u_{N+1} \leq u_N,sup{sn​∣n>N+1}≤sup{sn​∣n>N}⟹uN+1​≤uN​,

i.e., {uN}N\set{u_N}_N{uN​}N​ is a decreasing sequence. This tells you that lim sup⁡n→∞sn\limsup_{n\to\infty} s_nlimsupn→∞​sn​, which is equal to lim⁡N→∞uN\lim_{N\to\infty} u_NlimN→∞​uN​, always makes sense. If {uN}N\set{u_N}_N{uN​}N​ is bounded, then it converges by the earlier theorem. Otherwise, it diverges to −∞-\infty−∞. In the first case, this tells you that

lim sup⁡n→∞sn=inf⁡Nsup⁡ {sn∣n>N}.\limsup_{n\to\infty} s_n = \inf_N \sup\,\set{s_n \mid n > N}.n→∞limsup​sn​=Ninf​sup{sn​∣n>N}.

Similarly,

lim inf⁡n→∞sn=sup⁡Ninf⁡ {sn∣n>N}.\liminf_{n\to\infty} s_n = \sup_N \inf\,\set{s_n \mid n > N}.n→∞liminf​sn​=Nsup​inf{sn​∣n>N}.

Intuitively, lim sup⁡n→∞sn\limsup_{n\to\infty} s_nlimsupn→∞​sn​ is the "biggest thing sns_nsn​ can converge to" and lim inf⁡n→∞sn\liminf_{n\to\infty} s_nliminfn→∞​sn​ is the "smallest thing sns_nsn​ can converge to". For example:
