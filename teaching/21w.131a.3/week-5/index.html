1.9

This was the problem where you had to prove that 2n>n22^n > n^22n>n2 for all n≥5n \geq 5n≥5 by induction. I saw a lot of proofs that looked like this:

2n+1=2⋅2n>2n2>n2+2n+1(since n≥5)=(n+1)2.\begin{aligned} 2^{n+1} &= 2 \cdot 2^n \\ &> 2n^2 \\ &> n^2 + 2n + 1 && \p{\text{since } n \geq 5} \\ &= \p{n + 1}^2. \end{aligned}2n+1​=2⋅2n>2n2>n2+2n+1=(n+1)2.​​(since n≥5)​

I mainly had issue with the third line: while it is true that n2>2n+1n^2 > 2n + 1n2>2n+1 when n≥5n \geq 5n≥5, you still have to prove it on your homework. As a general rule of thumb, you should prove all claims you make that weren't proven in lecture, discussion, or the book (unless Professor Jacobs or I say it's okay to take something for granted). That being said, here is one way to prove it: since n≥5≥3n \geq 5 \geq 3n≥5≥3,

n2=n⋅n≥3n=2n+n>2n+1.n^2 = n \cdot n \geq 3n = 2n + n > 2n + 1.n2=n⋅n≥3n=2n+n>2n+1.

Miscellaneous

Let P,QP, QP,Q be propositional statements. If P  ⟹  QP \implies QP⟹Q is true and QQQ is true, does this mean PPP is also true? Hopefully, you agree that this is a false statement, and we can check with a logic table:

PQP  ⟹  Q111100011001\begin{array}{ccc} P & Q & P \implies Q \\\hline 1 & 1 & 1 \\ 1 & 0 & 0 \\ 0 & 1 & 1 \\ 0 & 0 & 1 \\ \end{array}P1100​Q1010​P⟹Q1011​​

The third row is the important row for this example: QQQ is true and P  ⟹  QP \implies QP⟹Q is true, but PPP is false. With this in mind, consider the following example:

Example 1.

What's the problem with the following argument?

Note that

n2−2n−1≥0  ⟹  (n−1)2−2≥0  ⟹  (n−1)2≥2.\begin{aligned} n^2 - 2n - 1 \geq 0 &\implies \p{n - 1}^2 - 2 \geq 0 \\ &\implies \p{n - 1}^2 \geq 2. \end{aligned}n2−2n−1≥0​⟹(n−1)2−2≥0⟹(n−1)2≥2.​

This is true if n≥3n \geq 3n≥3. Therefore, n2−2n−1≥0n^2 - 2n - 1 \geq 0n2−2n−1≥0 when n≥3n \geq 3n≥3.
Solution.

While the conclusion is true, the proof (which is the most important part) is incorrect: set P=‘‘n2−2n−1≥0"P = ``n^2 - 2n - 1 \geq 0"P=‘‘n2−2n−1≥0" and Q=‘‘(n−1)2≥2"Q = ``\p{n - 1}^2 \geq 2"Q=‘‘(n−1)2≥2", and we assume that n≥3n \geq 3n≥3. Then the incorrect proof is using the following logic:

P  ⟹  QP \implies QP⟹Q is true and QQQ is true. Therefore PPP is true.

As we discussed, this logic is not correct. Instead, you should do this in reverse: show that something true implies something else is true. In this case, the proof is correct if we just reverse the order of the steps: for n≥3n \geq 3n≥3,

(n−1)2≥2  ⟹  n2−2n+1≥2  ⟹  n2−2n−1≥0,\begin{aligned} \p{n - 1}^2 \geq 2 &\implies n^2 - 2n + 1 \geq 2 \\ &\implies n^2 - 2n - 1 \geq 0, \end{aligned}(n−1)2≥2​⟹n2−2n+1≥2⟹n2−2n−1≥0,​

and now you can conclude that n2−2n−1≥0n^2 - 2n - 1 \geq 0n2−2n−1≥0 for n≥3n \geq 3n≥3.

Limit Laws

Theorem (limit laws)

Suppose lim⁡n→∞an=a\displaystyle\lim_{n\to\infty} a_n = an→∞lim​an​=a and lim⁡n→∞bn=b\displaystyle \lim_{n\to\infty} b_n = bn→∞lim​bn​=b. Then:

  1. lim⁡n→∞(an+bn)=a+b\displaystyle\lim_{n\to\infty} \p{a_n + b_n} = a + bn→∞lim​(an​+bn​)=a+b.
  2. If c∈Rc \in \Rc∈R, then lim⁡n→∞can=ca\displaystyle\lim_{n\to\infty} ca_n = can→∞lim​can​=ca.
  3. lim⁡n→∞(anbn)=ab\displaystyle\lim_{n\to\infty} \p{a_nb_n} = abn→∞lim​(an​bn​)=ab.
  4. If b≠0b \neq 0b=0, then lim⁡n→∞anbn=ab\displaystyle\lim_{n\to\infty} \frac{a_n}{b_n} = \frac{a}{b}n→∞lim​bn​an​​=ba​.

This looks like a straightforward theorem, but it can be very easy to misuse it. The most common way I've seen it be used incorrectly is trying to apply it with limits that don't exist.

Example 2.

If lim⁡n→∞an\displaystyle\lim_{n\to\infty} a_nn→∞lim​an​ and lim⁡n→∞bn\displaystyle\lim_{n\to\infty} b_nn→∞lim​bn​ don't exist, does this mean an+bna_n + b_nan​+bn​ doesn't exist?
Solution.

The following is wrong:

lim⁡n→∞(an+bn)=lim⁡n→∞an+lim⁡n→∞bn\displaystyle\lim_{n\to\infty} \p{a_n + b_n} = \displaystyle\lim_{n\to\infty} a_n + \displaystyle\lim_{n\to\infty} b_nn→∞lim​(an​+bn​)=n→∞lim​an​+n→∞lim​bn​, and these limits don't exist. Therefore, lim⁡n→∞(an+bn)\displaystyle\lim_{n\to\infty} \p{a_n + b_n}n→∞lim​(an​+bn​) doesn't exist either.

The problem is with the first "=" sign: the limits don't exist, so that "=" is invalid. As a counterexample, consider an=(−1)na_n = \p{-1}^nan​=(−1)n and bn=−(−1)nb_n = -\p{-1}^nbn​=−(−1)n. Then both limits don't exist, but

lim⁡n→∞(an+bn)=lim⁡n→∞((−1)n−(−1)n)=lim⁡n→∞0=0,\lim_{n\to\infty} \p{a_n + b_n} = \lim_{n\to\infty} \p{\p{-1}^n - \p{-1}^n} = \lim_{n\to\infty} 0 = 0,n→∞lim​(an​+bn​)=n→∞lim​((−1)n−(−1)n)=n→∞lim​0=0,

so it's possible that the sum of two divergent sequences is convergent.

Another (very) useful theorem for calculating limits is the squeeze theorem:

Theorem (squeeze theorem)

Let {an}n\set{a_n}_n{an​}n​, {bn}n\set{b_n}_n{bn​}n​, and {sn}n\set{s_n}_n{sn​}n​ be sequences such that an≤sn≤bna_n \leq s_n \leq b_nan​≤sn​≤bn​. If lim⁡n→∞an=lim⁡n→∞bn=L\displaystyle\lim_{n\to\infty} a_n = \lim_{n\to\infty} b_n = Ln→∞lim​an​=n→∞lim​bn​=L, then lim⁡n→∞sn=L\displaystyle \lim_{n\to\infty} s_n = Ln→∞lim​sn​=L also.
Proof.

Let ε>0\epsilon > 0ε>0. Notice that

sn−L≤bn−L≤∣bn−L∣and−∣an−L∣≤an−L≤sn−L.s_n - L \leq b_n - L \leq \abs{b_n - L} \quad\text{and}\quad -\abs{a_n - L} \leq a_n - L \leq s_n - L.sn​−L≤bn​−L≤∣bn​−L∣and−∣an​−L∣≤an​−L≤sn​−L.

Since lim⁡n→∞an=L\displaystyle\lim_{n\to\infty} a_n = Ln→∞lim​an​=L, there exists N1∈RN_1 \in \RN1​∈R such that

n>N1  ⟹  ∣an−L∣<ε  ⟹  −ε<−∣an−L∣≤sn−L.\begin{aligned} n > N_1 &\implies \abs{a_n - L} < \epsilon \\ &\implies -\epsilon < -\abs{a_n - L} \leq s_n - L. \end{aligned}n>N1​​⟹∣an​−L∣<ε⟹−ε<−∣an​−L∣≤sn​−L.​

Similarly, because lim⁡n→∞bn=L\displaystyle\lim_{n\to\infty} b_n = Ln→∞lim​bn​=L, there exists N2∈RN_2 \in \RN2​∈R so that

n>N2  ⟹  ∣bn−L∣<ε  ⟹  sn−L≤∣bn−L∣<ε.\begin{aligned} n > N_2 &\implies \abs{b_n - L} < \epsilon \\ &\implies s_n - L \leq \abs{b_n - L} < \epsilon. \end{aligned}n>N2​​⟹∣bn​−L∣<ε⟹sn​−L≤∣bn​−L∣<ε.​

These inequalities work for different conditions on nnn, but we want to use them at the same time. If we let N=max⁡ {N1,N2}N = \max\,\set{N_1, N_2}N=max{N1​,N2​}, then whenever n>Nn > Nn>N, nnn satisfies n>N1n > N_1n>N1​ and n>N2n > N_2n>N2​ at the same time. Thus,

−ε<sn−L<ε  ⟹  ∣sn−L∣<ε.-\epsilon < s_n - L < \epsilon \implies \abs{s_n - L} < \epsilon.−ε<sn​−L<ε⟹∣sn​−L∣<ε.

□\square□

We generally use the following corollary of the squeeze theorem the most:

Corollary

Let {sn}n\set{s_n}_n{sn​}n​ and {tn}n\set{t_n}_n{tn​}n​ be sequences such that ∣sn∣≤tn\abs{s_n} \leq t_n∣sn​∣≤tn​ and lim⁡n→∞tn=0\displaystyle\lim_{n\to\infty} t_n = 0n→∞lim​tn​=0. Then lim⁡n→∞sn=0\displaystyle\lim_{n\to\infty} s_n = 0n→∞lim​sn​=0.
Proof.

Observe that −tn≤sn≤tn-t_n \leq s_n \leq t_n−tn​≤sn​≤tn​. Thus, if we set an=−tna_n = -t_nan​=−tn​ and bn=tnb_n = t_nbn​=tn​ as in the theorem, then an≤sn≤bna_n \leq s_n \leq b_nan​≤sn​≤bn​ and

lim⁡n→∞an=lim⁡n→∞bn=0  ⟹  lim⁡n→∞sn=0.\lim_{n\to\infty} a_n = \lim_{n\to\infty} b_n = 0 \implies \lim_{n\to\infty} s_n = 0.n→∞lim​an​=n→∞lim​bn​=0⟹n→∞lim​sn​=0.

□\square□

This corollary helps speeds up proofs quite a bit:

Example 4.

Prove that lim⁡n→∞∣n+2n2+1∣=0\displaystyle \lim_{n\to\infty} \abs{\frac{n + 2}{n^2 + 1}} = 0n→∞lim​∣∣​n2+1n+2​∣∣​=0.

In light of the squeeze theorem, we just need to find something bigger than ∣n+2n2+1∣\abs{\frac{n + 2}{n^2 + 1}}∣∣​n2+1n+2​∣∣​ which goes to 000:

Proving Divergence

As a quick example (relevant for your homework), here are two problems where you need to show that a limit doesn't exist:

Example 5.

Let a2n=1a_{2n} = 1a2n​=1 and a2n+1=3a_{2n+1} = 3a2n+1​=3. Show that lim⁡n→∞an\displaystyle\lim_{n\to\infty} a_nn→∞lim​an​ does not exist.
Solution.

Suppose otherwise and that lim⁡n→∞an=L\displaystyle\lim_{n\to\infty} a_n = Ln→∞lim​an​=L. Then given ε>0\epsilon > 0ε>0, there exists N∈RN \in \RN∈R such that whenever ∣an−L∣<ε\abs{a_n - L} < \epsilon∣an​−L∣<ε whenever n>Nn > Nn>N.

Suppose k>N2k > \frac{N}{2}k>2N​. Then

2k>N  ⟹  ∣a2k−L∣<ε.2k > N \implies \abs{a_{2k} - L} < \epsilon.2k>N⟹∣a2k​−L∣<ε.

Similarly,

2k+1>N  ⟹  ∣a2k+1−L∣<ε.2k + 1 > N \implies \abs{a_{2k+1} - L} < \epsilon.2k+1>N⟹∣a2k+1​−L∣<ε.

Based on these inequalities, if nnn is big enough, then LLL is within ε\epsilonε of 111 and 333. In the picture, this means that LLL has to be in both blue parentheses, and this should be impossible if ε\epsilonε is small enough. Based on the picture, we should get a contradiction if we let ε=1\epsilon = 1ε=1:

2=∣1−3∣=∣a2k−a2k+1∣≤∣a2k−L∣+∣a2k+1−L∣<1+1=2.2 = \abs{1 - 3} = \abs{a_{2k} - a_{2k+1}} \leq \abs{a_{2k} - L} + \abs{a_{2k+1} - L} < 1 + 1 = 2.2=∣1−3∣=∣a2k​−a2k+1​∣≤∣a2k​−L∣+∣a2k+1​−L∣<1+1=2.

This is impossible, so no LLL could have existed to begin with, which completes the proof.
Example 6.

Let an=2na_n = 2^nan​=2n. Prove that lim⁡n→∞2n\displaystyle\lim_{n\to\infty} 2^nn→∞lim​2n does not exist.
Solution.

Like before, we're going to assume (for the sake of contradiction) that lim⁡n→∞an=L\displaystyle\lim_{n\to\infty} a_n = Ln→∞lim​an​=L.

We're going to apply the same strategy: we're going to pick ε\epsilonε small enough so that LLL would have to be in two different places at the same time, which should give us a contradiction. Based on the picture, the distance between 2n2^n2n and 2n+12^{n+1}2n+1 is 2n≥22^n \geq 22n≥2, so ε=1\epsilon = 1ε=1 should work again.

By definition of the limit, there exists N∈RN \in \RN∈R such that if n>Nn > Nn>N, then ∣an−L∣<1\abs{a_n - L} < 1∣an​−L∣<1. Notice this means that ∣an+1−L∣<1\abs{a_{n+1} - L} < 1∣an+1​−L∣<1 as well, so

2n=∣an−an+1∣≤∣an−L∣+∣an+1−L∣<1+1=2.2^n = \abs{a_n - a_{n+1}} \leq \abs{a_n - L} + \abs{a_{n+1} - L} < 1 + 1 = 2.2n=∣an​−an+1​∣≤∣an​−L∣+∣an+1​−L∣<1+1=2.

Thus, we get a contradiction, so the limit must not have existed to begin with.
