Supremums and Infimums

Definition

Let S⊆RS \subseteq \RS⊆R be a non-empty set. If there exists x∈Sx \in Sx∈S such that s≤xs \leq xs≤x for all s∈Ss \in Ss∈S, then we call xxx the maximum of SSS and we write x=max⁡Sx = \max{S}x=maxS.
Exercise 1.

Define min⁡S\min{S}minS.

At first glance, max⁡S\max{S}maxS and sup⁡S\sup{S}supS look like the same thing. However, there's a key difference: max⁡S\max{S}maxS has to be an element of SSS, but sup⁡S\sup{S}supS doesn't have to be.

Example 1.

If S=[0,1)S = \pco{0, 1}S=[0,1), then SSS has no maximum, but sup⁡S=1\sup{S} = 1supS=1.

On the other hand, if S=[0,1]S = \br{0, 1}S=[0,1], then max⁡S=sup⁡S=1\max{S} = \sup{S} = 1maxS=supS=1.
Example 2.

(4.8) Let SSS and TTT be non-empty subsets of R\RR such that s≤ts \leq ts≤t for all s∈Ss \in Ss∈S and t∈Tt \in Tt∈T.

  1. Observe that SSS is bounded above and TTT is bounded below.
  2. Prove sup⁡S≤inf⁡T\sup{S} \leq \inf{T}supS≤infT.
Solution.
  1. Any element of TTT is an upper bound of SSS, and any element of SSS is a lower bound of TTT.
  2. If t∈Tt \in Tt∈T, then ttt is an upper bound for SSS, so by definition, sup⁡S≤t\sup{S} \leq tsupS≤t. This means that sup⁡S\sup{S}supS is a lower bound for TTT, so sup⁡S≤inf⁡T\sup{S} \leq \inf{T}supS≤infT.

Density of Q\QQ in R\RR

Theorem

For any a,b∈Ra, b \in Ra,b∈R with a<ba < ba<b, there exists a rational q∈Qq \in \Qq∈Q such that q∈(a,b)q \in \p{a, b}q∈(a,b).

We know that Q\QQ has gaps in it, which is why we work with R\RR. However, the fact that Q\QQ is dense means that the gaps are very "tiny". In other words, no matter how far you zoom in, you'll always see infinitely many rational numbers.

Example 3.

Solution.

Sequences

Definition

A sequence is a function s ⁣:N→R\func{s}{\N}{\R}s:N→R, where we write sn=s(n)s_n = s\p{n}sn​=s(n) and {sn}n={sn∣n∈N}\set{s_n}_n = \set{s_n \mid n \in \N}{sn​}n​={sn​∣n∈N}.

If there exists L∈RL \in \RL∈R such that:

For all ε>0\epsilon > 0ε>0, there exists N∈RN \in \RN∈R such that ∣sn−L∣<ε\abs{s_n - L} < \epsilon∣sn​−L∣<ε for all N>NN > NN>N,

we say that sns_nsn​ converges to LLL and we write lim⁡n→∞=L\displaystyle \lim_{n\to\infty} = Ln→∞lim​=L. Otherwise, if no such LLL exists, then we say that sns_nsn​ is divergent.

Intuitively, the limit of a sequence is what it gets "really close to" as nnn "gets really big". To make this more precise, we need to quantify two things:

  • "really close to" (this is ε\epsilonε)
  • "gets really big" (this is NNN)

With this interpretation, you can read the definition like this:

If nnn is large enough, we can make ∣sn−L∣\abs{s_n - L}∣sn​−L∣ as small as we want.

The definition is (somewhat) intuitive, but using the definition to prove things is generally pretty tricky.

Example 4.

(Similar to 8.1(c)) Prove lim⁡n→∞4n+37n−5=47\displaystyle \lim_{n\to\infty} \frac{4n+3}{7n-5} = \frac{4}{7}n→∞lim​7n−54n+3​=74​.
Solution.

When proving limits, we're given an ε>0\epsilon > 0ε>0 (something out of your control), and we need to find NNN (something you can control). The general strategy is to work backwards: start with

∣4n+37n−5−47∣=∣21+207(7n−5)∣=417(7n−5)\abs{\frac{4n+3}{7n-5} - \frac{4}{7}} = \abs{\frac{21 + 20}{7\p{7n-5}}} = \frac{41}{7\p{7n-5}}∣∣​7n−54n+3​−74​∣∣​=∣∣​7(7n−5)21+20​∣∣​=7(7n−5)41​

when n≥1n \geq 1n≥1. In the end, we want to make this quantity smaller than ε\epsilonε, so it's okay to look at bigger quantities. If we make the denominator smaller, then we get something bigger:

∣417(7n−5)∣≤417(7n−5n)=4114n.\abs{\frac{41}{7\p{7n-5}}} \leq \frac{41}{7\p{7n - 5n}} = \frac{41}{14n}.∣∣​7(7n−5)41​∣∣​≤7(7n−5n)41​=14n41​.

If we can make 4114n<ε\frac{41}{14n} < \epsilon14n41​<ε, then we're done. We're in good shape because we can:

4114n<ε  ⟺  n>4114ε.\frac{41}{14n} < \epsilon \iff n > \frac{41}{14\epsilon}.14n41​<ε⟺n>14ε41​.

None of what I wrote above is a proof (though it almost looks like one); it's just scratch work that I used to figure out what NNN needs to be. Given an ε>0\epsilon > 0ε>0, our candidate for NNN is 4114ε\frac{41}{14\epsilon}14ε41​, and it's now time to verify that this works. The following is the beginning of the proof of the limit:

Let ε>0\epsilon > 0ε>0. If we set N=4114εN = \frac{41}{14\epsilon}N=14ε41​, then if n>Nn > Nn>N,

∣sn−L∣=∣4n+37n−5−47∣=417(7n−5)≤4114n<4114N=4114(41/14ε)=ε.\begin{aligned} \abs{s_n - L} = \abs{\frac{4n+3}{7n-5} - \frac{4}{7}} = \frac{41}{7\p{7n-5}} &\leq \frac{41}{14n} \\ &< \frac{41}{14N} \\ &= \frac{41}{14\p{41/14\epsilon}} \\ &= \epsilon. \end{aligned}∣sn​−L∣=∣∣​7n−54n+3​−74​∣∣​=7(7n−5)41​​≤14n41​<14N41​=14(41/14ε)41​=ε.​

Example 5.

(Similar to 8.1(d)) Prove lim⁡n→∞n+3n2−3=0\displaystyle \lim_{n\to\infty} \frac{n+3}{n^2-3} = 0n→∞lim​n2−3n+3​=0.
Solution.

We're going to apply the same strategy and work backwards: if n>2n > 2n>2, then

∣n+3n2−3∣=n+3n2−3≤n+3n2−9=n+3(n+3)(n−3)=1n−3.\abs{\frac{n + 3}{n^2 - 3}} = \frac{n + 3}{n^2 - 3} \leq \frac{n + 3}{n^2 - 9} = \frac{n + 3}{\p{n + 3}\p{n - 3}} = \frac{1}{n - 3}.∣∣​n2−3n+3​∣∣​=n2−3n+3​≤n2−9n+3​=(n+3)(n−3)n+3​=n−31​.

So N=3+1εN = 3 + \frac{1}{\epsilon}N=3+ε1​ should work. We just need to prove that this works:

Let ε>0\epsilon > 0ε>0. If N=3+1εN = 3 + \frac{1}{\epsilon}N=3+ε1​, then for n>Nn > Nn>N, we get

∣sn−L∣=∣n+3n2−3∣=n+3n2−3≤n+3n2−9=1n−3<1N−3=13+1ε−3=ε.\begin{aligned} \abs{s_n - L} = \abs{\frac{n + 3}{n^2 - 3}} = \frac{n + 3}{n^2 - 3} &\leq \frac{n + 3}{n^2 - 9} \\ &= \frac{1}{n - 3} \\ &< \frac{1}{N - 3} \\ &= \frac{1}{3 + \frac{1}{\epsilon} - 3} \\ &= \epsilon. \end{aligned}∣sn​−L∣=∣∣​n2−3n+3​∣∣​=n2−3n+3​​≤n2−9n+3​=n−31​<N−31​=3+ε1​−31​=ε.​

To summarize, this is the basic strategy for proving limits through the definition:

  1. Work backwards and use inequalities figure out a candidate for NNN.
    • Generally, you want to make the numerator bigger (in absolute value) and/or the denominator smaller (in absolute value).
  2. Write out the proof with your candidate of NNN.
    • If you set it up right, then your proof will just look like what you wrote in step 1, but in reverse.

The definition can also be used to prove some useful facts about limits.

Proposition

Let {sn}n\set{s_n}_n{sn​}n​ be a sequence. Suppose sn>0s_n > 0sn​>0 for all n≥1n \geq 1n≥1 and that L=lim⁡n→∞snL = \displaystyle\lim_{n\to\infty} s_nL=n→∞lim​sn​ exists. Then L≥0L \geq 0L≥0.
Proof.

Intuitively, it's obvious. However, proving it is a bit tricky since we need to use the definition of a limit. We'll do this by contradiction: assume for the sake of contradiction that L<0L < 0L<0. We then have the following picture:

The definition of a limit tells you that for any ε>0\epsilon > 0ε>0, eventually all the points in your sequence lie in the interval (L−ε,L+ε)\p{L - \epsilon, L + \epsilon}(L−ε,L+ε). From the picture, you should be able to see why that's impossible: if ε\epsilonε is small enough, then that implies that the sequence is eventually negative, which is a contradiction. Now we need to write that down:

If ε\epsilonε is too large, then we won't get a contradiction. Thus, we need to be smart about how we pick it. Based on the picture, if we let ε\epsilonε be half the distance from LLL to 000, then we should be able to get a contradiction, i.e., let ε=∣L∣2=−L2>0\epsilon = \frac{\abs{L}}{2} = -\frac{L}{2} > 0ε=2∣L∣​=−2L​>0. By definition of the limit, there exists N∈RN \in \RN∈R such that if n>Nn > Nn>N, then ∣sn−L∣<ε\abs{s_n - L} < \epsilon∣sn​−L∣<ε. But this means

−ε<sn−L<ε  ⟹  sn<L+ε=L−L2=L2<0,-\epsilon < s_n - L < \epsilon \implies s_n < L + \epsilon = L - \frac{L}{2} = \frac{L}{2} < 0,−ε<sn​−L<ε⟹sn​<L+ε=L−2L​=2L​<0,

which is impossible, so L≥0L \geq 0L≥0 to begin with. □\square□
