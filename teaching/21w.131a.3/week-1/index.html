Set Theory

Common Sets

Here are some very common sets that we'll be using throughout the class:

  • ∅ =\emptyset\: =∅= "empty set"
  • N={1,2,3,…}=\N = \set{1, 2, 3, \ldots} =N={1,2,3,…}= "natural numbers" (in our textbook, natural numbers start at 111, but that might not be the case in other books)
  • Z={…,−2,−1,0,1,2,…}=\Z = \set{\ldots, -2, -1, 0, 1, 2, \ldots} =Z={…,−2,−1,0,1,2,…}= "integers"
  • 2Z={2n∣n∈Z}=2\Z = \set{2n \mid n \in \Z} =2Z={2n∣n∈Z}= "even integers"
  • Q=\Q =Q= "rational numbers"
  • R=\R =R= "real numbers"

We will be covering the construction of these sets (especially Q\QQ and R\RR) in great detail a little later in the class, but these are the sets that you "know" already.

Remark.

∅\emptyset∅ is a subset of every set. The idea goes like this: if ∅\emptyset∅ were not a subset of AAA, then there exists x∈∅x \in \emptysetx∈∅ such that x∉Ax \notin Ax∈/A. But ∅\emptyset∅ has no elements, so ∅⊆A\emptyset \subseteq A∅⊆A. This is an example of a vacuous truth.

Set Operations

Definition

Let A,BA, BA,B be sets. Then

  1. The union of AAA and BBB is A∪B={x∣x∈A or x∈B}A \cup B = \set{x \mid x \in A \text{ or } x \in B}A∪B={x∣x∈A or x∈B}.
  2. The intersection of AAA and BBB is A∩B={x∣x∈A and x∈B}A \cap B = \set{x \mid x \in A \text{ and } x \in B}A∩B={x∣x∈A and x∈B}.
  3. The complement of AAA is Ac={x∣x∉A}A^\comp = \set{x \mid x \notin A}Ac={x∣x∈/A}.

Let's understand these one-by-one:

  1. Taking the union of AAA and BBB just means dumping everything in AAA and everything in BBB into one bigger set.
  2. The intersection of AAA and BBB means everything that's in both sets.
  3. The complement of AAA is everything that's outside of AAA (i.e., everything that's not an element of AAA).

Now that we have some basic set operations, it's natural to ask what happens when we mix them together:

Proposition (De Morgan's laws)

Let A,BA, BA,B be sets. Then

  1. (A∪B)c=Ac∩Bc\p{A \cup B}^\comp = A^\comp \cap B^\comp(A∪B)c=Ac∩Bc
  2. (A∩B)c=Ac∪Bc\p{A \cap B}^\comp = A^\comp \cup B^\comp(A∩B)c=Ac∪Bc
Proof.

(i): If x∈A∪Bx \in A \cup Bx∈A∪B, then xxx is in at least one of AAA or BBB. The negation of "at least one" is "none," so x∉A∪Bx \notin A \cup Bx∈/A∪B if and only if x∉Ax \notin Ax∈/A and x∉Bx \notin Bx∈/B, which means

(A∪B)c={x∣x∉A and x∉B}=Ac∩Bc.\p{A \cup B}^\comp = \set{x \mid x \notin A \text{ and } x \notin B} = A^\comp \cap B^\comp.(A∪B)c={x∣x∈/A and x∈/B}=Ac∩Bc.

Here's the picture:

(ii): The idea is the same: if x∈A∩Bx \in A \cap Bx∈A∩B, then xxx is in both AAA and BBB. The negation of "both" is "at most one," so x∉A∩Bx \notin A \cap Bx∈/A∩B if and only if x∉Ax \notin Ax∈/A or x∉Bx \notin Bx∈/B. This means

(A∩B)c={x∣x∉A or x∉B}=Ac∪Bc.\p{A \cap B}^\comp = \set{x \mid x \notin A \text{ or } x \notin B} = A^\comp \cup B^\comp.(A∩B)c={x∣x∈/A or x∈/B}=Ac∪Bc.

□\square□

Definition

Let A,BA, BA,B be sets. Then

  1. The power set of AAA is P(A)={B∣B⊆A}\pow\p{A} = \set{B \mid B \subseteq A}P(A)={B∣B⊆A}.
  2. The cartesian product of AAA and BBB is A×B={(a,b)∣a∈A and x∈B}A \times B = \set{\p{a, b} \mid a \in A \text{ and } x \in B}A×B={(a,b)∣a∈A and x∈B}.

Like before, here's a less mathy way of looking at these sets:

  1. The power set of AAA is the set of subsets of AAA, so it's a set of sets.
  2. The cartesian product of AAA and BBB just means you pick one element of AAA and one element of BBB and put them in a pair.
Example 1.

If A={0,1,2}A = \set{0, 1, 2}A={0,1,2}, then

P(A)={∅,{0},{1},{2},{0,1},{0,2},{1,2},{0,1,2}}.\pow\p{A} = \set{\emptyset, \set{0}, \set{1}, \set{2}, \set{0, 1}, \set{0, 2}, \set{1, 2}, \set{0, 1, 2}}.P(A)={∅,{0},{1},{2},{0,1},{0,2},{1,2},{0,1,2}}.

If A={∅,1}A = \set{\emptyset, 1}A={∅,1}, then

P(A)={∅,{∅},{1},{∅,1}}.\pow\p{A} = \set{\emptyset, \set{\emptyset}, \set{1}, \set{\emptyset, 1}}.P(A)={∅,{∅},{1},{∅,1}}.

Counterexamples

It's good to know a lot of counterexamples to prevent you from writing down things that are wrong. A lot of times, something is "intuitively true," but turns out to be false. For example, when I took Calc AB in high school, I wrote something like this:

Since fff is continuous, it is differentiable somewhere.

I now know how wrong I was, but as a beginner, this made sense to me since it was true for every example I could think of. The issue was that I wasn't exposed to enough counterexamples. This statement is false because there are functions which are continuous everywhere, but differentiable nowhere.

To see why fnf_nfn​ converges to 000, you can slide x0x_0x0​ to be any number in [0,1]\br{0, 1}[0,1]. You'll see that if nnn is large enough, then the spike won't include x=x0x = x_0x=x0​. Example 6.

True or false: If fff is continuous, then the image of (a,b)\p{a, b}(a,b) is still an interval. (An interval is anything of the form (c,d),[c,d),(c,d]\p{c, d}, \pco{c, d}, \poc{c, d}(c,d),[c,d),(c,d], or [c,d]\br{c, d}[c,d].)
Solution.

This is true because of the intermediate value theorem (it's a bit technical to explain in detail, though, but we'll revisit this in the future). This is also related to something called connectedness, which is covered in MATH 131B.

The main takeaway from these examples is that there is bad intuition and good intuition. Knowing counterexamples helps you get rid of the bad intuition, which helps you avoid making mistakes.
