11.11

I saw a lot of people write something like this:

By Theorem 11.4, every sequence has an increasing subsequence.

Here is the statement of Theorem 11.4:

Every sequence {sn}n\set{s_n}_n{sn​}n​ has a monotonic subsequence.

Notice that the theorem only says that every sequence has a monotonic sequence, which is why the statement is false. It's possible for a sequence to have no increasing subsequences, e.g., if sn=1ns_n = \frac{1}{n}sn​=n1​, then every subsequence of {sn}n\set{s_n}_n{sn​}n​ is decreasing. However, in this problem, you can show that the monotone sequence you get does actually increase.

I saw the following (incorrect) statements pop up a bunch of times:

lim sup⁡n→∞sn=0  ⟹  lim sup⁡n→∞ ∣sn∣=0\displaystyle\limsup_{n\to\infty} s_n = 0 \implies \limsup_{n\to\infty} \,\abs{s_n} = 0n→∞limsup​sn​=0⟹n→∞limsup​∣sn​∣=0

∣sup⁡ {x∣x∈S}∣≠sup⁡ {∣x∣∣x∈S}.\abs{\sup\,\set{x \mid x \in S}} \neq \sup\,\set{\abs{x} \mid x \in S}.∣sup{x∣x∈S}∣=sup{∣x∣∣x∈S}.

As for a counterexample for this specific statement, the sequence sn=−1+(−1)ns_n = -1 + \p{-1}^nsn​=−1+(−1)n works. This is the sequence that alternates between 000 and −2-2−2, so

lim sup⁡n→∞sn=0butlim sup⁡n→∞ ∣sn∣=2.\limsup_{n\to\infty} s_n = 0 \quad\text{but}\quad \limsup_{n\to\infty} \,\abs{s_n} = 2.n→∞limsup​sn​=0butn→∞limsup​∣sn​∣=2.

I also saw something like:

lim sup⁡n→∞ ∣sn∣=−lim inf⁡n→∞ ∣−sn∣\displaystyle\limsup_{n\to\infty} \,\abs{s_n} = -\liminf_{n\to\infty} \,\abs{-s_n}n→∞limsup​∣sn​∣=−n→∞liminf​∣−sn​∣

lim sup⁡n→∞tn=−lim inf⁡n→∞ (−tn).\limsup_{n\to\infty} t_n = -\liminf_{n\to\infty} \,\p{-t_n}.n→∞limsup​tn​=−n→∞liminf​(−tn​).

lim sup⁡n→∞ ∣sn∣=−lim inf⁡n→∞ (−∣sn∣),\limsup_{n\to\infty} \,\abs{s_n} = -\liminf_{n\to\infty} \,\p{-\abs{s_n}},n→∞limsup​∣sn​∣=−n→∞liminf​(−∣sn​∣),

i.e., the negative sign goes outside of the absolute values. As a counterexample, the sequence sn=1s_n = 1sn​=1 works:

Continuous Functions

Since things don't seem to be too hard this week, I decided to do a fun example.

Example 1.

Let f ⁣:R→R\func{f}{\R}{\R}f:R→R be a continuous function such that f(1)=1f\p{1} = 1f(1)=1 such that f(1)=1f\p{1} = 1f(1)=1 and f(nx)=nf(x)f\p{nx} = nf\p{x}f(nx)=nf(x) for any n∈Zn \in \Zn∈Z and x∈Rx \in \Rx∈R. Then f(x)=xf\p{x} = xf(x)=x for all x∈Rx \in \Rx∈R.
Solution.

The idea here is that it's hard to show directly for any x∈Rx \in \Rx∈R that this equality is true, since it's hard to connect irrational numbers to the conditions given to you. Instead, however, we can show that it's true for x∈Qx \in \Qx∈Q, and by a proposition from last week, it follows that f(x)=xf\p{x} = xf(x)=x everywhere.

First, if we set x=1x = 1x=1, then for n∈Zn \in \Zn∈Z, we get

f(n)=f(n⋅1)=nf(1)=n.f\p{n} = f\p{n \cdot 1} = nf\p{1} = n.f(n)=f(n⋅1)=nf(1)=n.

Next, we can let x=1nx = \frac{1}{n}x=n1​ to get

f(1)=f(n⋅1n)=nf(1n)  ⟹  f(1n)=1n.f\p{1} = f\p{n \cdot \frac{1}{n}} = nf\p{\frac{1}{n}} \implies f\p{\frac{1}{n}} = \frac{1}{n}.f(1)=f(n⋅n1​)=nf(n1​)⟹f(n1​)=n1​.

Finally, if m,n∈Nm, n \in \Nm,n∈N, then

f(mn)=mf(1n)=mn.f\p{\frac{m}{n}} = mf\p{\frac{1}{n}} = \frac{m}{n}.f(nm​)=mf(n1​)=nm​.

Thus, f(q)=qf\p{q} = qf(q)=q for all q∈Qq \in \Qq∈Q, so by density, we have f(x)=xf\p{x} = xf(x)=x for all x∈Rx \in \Rx∈R.

Properties

Definition

Let f ⁣:D→R\func{f}{D}{\R}f:D→R be a function. Then the image of a set A⊆DA \subseteq DA⊆D under fff is f(A)={f(x)∣x∈A}f\p{A} = \set{f\p{x} \mid x \in A}f(A)={f(x)∣x∈A}.

Basically, the image tells you all the points that fff attains when you restrict the domain to AAA.

Let f(x)=x2f\p{x} = x^2f(x)=x2. Then

  • f(R)=[0,∞)f\p{\R} = \pco{0, \infty}f(R)=[0,∞)
  • f({−1,0,1,2,3})={0,1,4,9}f\p{\set{-1, 0, 1, 2, 3}} = \set{0, 1, 4, 9}f({−1,0,1,2,3})={0,1,4,9}

Next are two very important theorems about continuous functions:

Theorem (extreme value theorem)

If f ⁣:[a,b]→R\func{f}{\br{a,b}}{\R}f:[a,b]→R is continuous, then fff is bounded and there exist s0,s1∈[a,b]s_0, s_1 \in \br{a, b}s0​,s1​∈[a,b] such that f(s0)≤f(x)≤f(s1)f\p{s_0} \leq f\p{x} \leq f\p{s_1}f(s0​)≤f(x)≤f(s1​) for all x∈[a,b]x \in \br{a, b}x∈[a,b].

In short, this tells you that on a closed and bounded interval, fff is bounded and attains both its minimum and maximum. (In general, we say that the image of a compact set under a continuous function is compact, but this is beyond the scope of this class.)

Theorem (intermediate value theorem)

Let I⊆RI \subseteq \RI⊆R be an interval. If f ⁣:I→R\func{f}{I}{\R}f:I→R is a continuous function, then f(I)f\p{I}f(I) is an interval.

This basically tells you that continuous functions can't "jump" on an interval. (In general, the continuous image of a connected set is connected, but like compactness, this is beyond our class.)

Uniform Continuity

Definition

Let f ⁣:D→R\func{f}{D}{\R}f:D→R. Then fff is uniformly continuous on DDD if for all ε>0\epsilon > 0ε>0, there exists δ>0\delta > 0δ>0 such that ∣f(x)−f(y)∣<ε\abs{f\p{x} - f\p{y}} < \epsilon∣f(x)−f(y)∣<ε whenever x,y∈Dx, y \in Dx,y∈D satisfy ∣x−y∣<δ\abs{x - y} < \delta∣x−y∣<δ.
Remark.

Compare this with our usual definition of continuity:

For all x∈Dx \in Dx∈D and ε>0\epsilon > 0ε>0, there exists δ>0\delta > 0δ>0 such that ∣f(x)−f(y)∣<ε\abs{f\p{x} - f\p{y}} < \epsilon∣f(x)−f(y)∣<ε when y∈Dy \in Dy∈D satisfies ∣x−y∣<δ\abs{x - y} < \delta∣x−y∣<δ.

Notice that δ\deltaδ may depend on xxx, i.e., given a different x∈Dx \in Dx∈D, you may have to pick a different δ\deltaδ for regular continuity. However, for uniform continuity, you can find a δ\deltaδ that works for all values of x∈Dx \in Dx∈D.

This concept is useful when we need to bound a lot of values of fff at the same time, e.g., when finding inequalities for integrals or sums relating to continuous functions.

True or False

Example 6.

True or false: If fff and ggg are uniformly continuous on R\RR, then so is fgfgfg.
Solution.

False. If f(x)=g(x)=xf\p{x} = g\p{x} = xf(x)=g(x)=x, then f(x)g(x)=x2f\p{x}g\p{x} = x^2f(x)g(x)=x2, which is not uniformly continuous on R\RR.
Example 7.

True or false: If fff and ggg are uniformly continuous on R\RR, then so is f+gf + gf+g.
Solution.

True. Let ε>0\epsilon > 0ε>0 and notice that by the triangle inequality,

∣(f+g)(x)−(f+g)(y)∣=∣(f(x)−f(y))+(g(x)−g(y))∣≤∣f(x)−f(y)∣+∣g(x)−g(y)∣.\begin{aligned} \abs{\p{f + g}\p{x} - \p{f + g}\p{y}} &= \abs{\p{f\p{x} - f\p{y}} + \p{g\p{x} - g\p{y}}} \\ &\leq \abs{f\p{x} - f\p{y}} + \abs{g\p{x} - g\p{y}}. \end{aligned}∣(f+g)(x)−(f+g)(y)∣​=∣(f(x)−f(y))+(g(x)−g(y))∣≤∣f(x)−f(y)∣+∣g(x)−g(y)∣.​

Since fff is uniformly continuous on R\RR, there exists δ1>0\delta_1 > 0δ1​>0 such that if ∣x−y∣<δ1\abs{x - y} < \delta_1∣x−y∣<δ1​, then ∣f(x)−f(y)∣<ε2\abs{f\p{x} - f\p{y}} < \frac{\epsilon}{2}∣f(x)−f(y)∣<2ε​. Similarly, because ggg is uniformly continuous on R\RR, there exists δ2>0\delta_2 > 0δ2​>0 so that if ∣x−y∣<δ2\abs{x - y} < \delta_2∣x−y∣<δ2​, then ∣g(x)−g(y)∣<ε2\abs{g\p{x} - g\p{y}} < \frac{\epsilon}{2}∣g(x)−g(y)∣<2ε​.

Let δ=min⁡ {δ1,δ2}\delta = \min\,\set{\delta_1, \delta_2}δ=min{δ1​,δ2​}, so if ∣x−y∣<δ\abs{x - y} < \delta∣x−y∣<δ, then ∣x−y∣<δ1\abs{x - y} < \delta_1∣x−y∣<δ1​ and ∣x−y∣<δ2\abs{x - y} < \delta_2∣x−y∣<δ2​, and so

∣(f+g)(x)−(f+g)(y)∣≤∣f(x)−f(y)∣+∣g(x)−g(y)∣<ε2+ε2=ε,\abs{\p{f + g}\p{x} - \p{f + g}\p{y}} \leq \abs{f\p{x} - f\p{y}} + \abs{g\p{x} - g\p{y}} < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon,∣(f+g)(x)−(f+g)(y)∣≤∣f(x)−f(y)∣+∣g(x)−g(y)∣<2ε​+2ε​=ε,
