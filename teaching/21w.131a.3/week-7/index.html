Problem 2

Use induction to prove that 2n≤(n+1)!2^n \leq \p{n+1}!2n≤(n+1)! for all n≥0n \geq 0n≥0.

Solution.

Base case: (n=0n = 0n=0)

When n=0n = 0n=0, we get 20=1=1!2^0 = 1 = 1!20=1=1!, so the base case holds.

Inductive step:

Suppose 2n≤(n+1)!2^n \leq \p{n+1}!2n≤(n+1)!. We want to show that 2n+1≤(n+2)!2^{n+1} \leq \p{n+2}!2n+1≤(n+2)! also. First, notice that since n≥0n \geq 0n≥0, we get n+2≥2n + 2 \geq 2n+2≥2. Thus,

2n+1=2⋅2n≤(n+2)⋅(n+1)!=(n+2)!.2^{n+1} = 2 \cdot 2^n \leq \p{n+2} \cdot \p{n+1}! = \p{n+2}!.2n+1=2⋅2n≤(n+2)⋅(n+1)!=(n+2)!.

By induction, the inequality holds for all n≥0n \geq 0n≥0.

Definition

m≤n1<n2<⋯m \leq n_1 < n_2 < \cdotsm≤n1​<n2​<⋯

Example 1.

If sn=(−1)ns_n = \p{-1}^nsn​=(−1)n, then s2k=1s_{2k} = 1s2k​=1 and s2k+1=−1s_{2k+1} = -1s2k+1​=−1.

Subsequences are similar to subsets, but with one important distinction: you have to respect the ordering of the original sequence. For example, if sn=ns_n = nsn​=n, then

1,3,2,4,5,…1, 3, 2, 4, 5, \ldots1,3,2,4,5,…

is not a subsequence of {sn}n\set{s_n}_n{sn​}n​.

Theorem (Bolzano-Weierstrass)

Every bounded sequence has a convergent subsequence.

This is a very important idea in analysis, but it's won't be particularly useful for us while we're looking at regular sequences.

Theorem

A sequence {sn}n\set{s_n}_n{sn​}n​ converges if and only if every subsequence of {sn}n\set{s_n}_n{sn​}n​ converges to the same limit.

This theorem is useful for us because it gives us another way to prove that sequence diverges. Instead of using the ε\epsilonε definition of a limit, we can just find two subsequences which converge to different numbers.
