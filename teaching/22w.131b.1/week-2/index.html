Metric Space Topology

Open Balls

Definition

Let (X,d)\p{X, d}(X,d) be a metric space. If x∈Xx \in Xx∈X and r>0r > 0r>0, then the open ball of radius rrr centered at xxx is

B(x,r)={y∈X∣d(x,y)<r}.B\p{x, r} = \set{y \in X \mid d\p{x, y} < r}.B(x,r)={y∈X∣d(x,y)<r}.

When talking about topology (i.e., open sets, closed sets, etc.), open balls are like the "atoms." They're going to be the main object used in the definitions and concepts of this section.

Definition

Let (X,d)\p{X, d}(X,d) be a metric space and E⊆XE \subseteq XE⊆X be a subset.

  1. The interior of EEE, denoted Int⁡E\Int{E}IntE, is the set of points x∈Xx \in Xx∈X where there exists r>0r > 0r>0 such that B(x,r)⊆EB\p{x, r} \subseteq EB(x,r)⊆E.
  2. The exterior of EEE, denoted Ext⁡E\Ext{E}ExtE, is the set of points x∈Xx \in Xx∈X where there exists r>0r > 0r>0 such that B(x,r)⊆X∖EB\p{x, r} \subseteq X \setminus EB(x,r)⊆X∖E.
  3. The boundary of EEE, denoted ∂E\bd{E}∂E, is the set of points that are not in Int⁡E\Int{E}IntE or Ext⁡E\Ext{E}ExtE.

Open and Closed Sets

Definition

Let (X,d)\p{X, d}(X,d) be a metric space and E⊆XE \subseteq XE⊆X be a subset.

  1. EEE is open if for every x∈Ex \in Ex∈E, there exists r>0r > 0r>0 such that B(x,r)⊆EB\p{x, r} \subseteq EB(x,r)⊆E.
  2. EEE is closed if its complement X∖EX \setminus EX∖E is open.

One thing to note that is open is not the opposite of closed.

The next two propositions tell you how open/closed sets interact with unions/intersections:

  1. ∅,X\emptyset, X∅,X are both open.

  2. If {Ui}i∈I\set{U_i}_{i \in I}{Ui​}i∈I​ is a collection of open sets indexed by III, then its union ⋃i∈IUi\bigcup_{i \in I} U_i⋃i∈I​Ui​ is also open.

  3. If U1,…,UnU_1, \ldots, U_nU1​,…,Un​ are open sets, then their intersection U1∩⋯∩UnU_1 \cap \cdots \cap U_nU1​∩⋯∩Un​ is also open.

  1. ∅,X\emptyset, X∅,X are both closed.

  2. If {Fi}i∈I\set{F_i}_{i \in I}{Fi​}i∈I​ is a collection of closed sets indexed by III, then its union ⋂i∈IFi\bigcap_{i \in I} F_i⋂i∈I​Fi​ is also closed.

The takeaway from these propositions is this:

  1. Open sets interact nicely with unions.
  2. Closed sets interact nicely with intersections.

Adherent Points and Closure

Definition

Let (X,d)\p{X, d}(X,d) be a metric space and E⊆XE \subseteq XE⊆X be a subset. Then x∈Xx \in Xx∈X (note that xxx is not required to be an element of EEE) is an adherent point of EEE if for every r>0r > 0r>0, we have B(x,r)∩E≠∅B\p{x, r} \cap E \neq \emptysetB(x,r)∩E=∅. The set of adherent points of EEE is called the closure of EEE and is denoted E‾\cl{E}E.

Let's take another look at the definition of an adherent point:

for all r>0, B(x,r)∩E≠∅.\text{for all } r > 0,\ B\p{x, r} \cap E \neq \emptyset.for all r>0, B(x,r)∩E=∅.

We can write it like this:

for all r>0, there exists y∈E such that d(x,y)<r.\text{for all } r > 0, \text{ there exists } y \in E \text{ such that } d\p{x, y} < r.for all r>0, there exists y∈E such that d(x,y)<r.

When you look at this definition, you should be thinking of what happens when rrr is really small. Then you can interpret an adherent point as follows:

‘‘x is an adherent point of E if it can be approximated by elements in E."``x \text{ is an adherent point of } E \text{ if it can be approximated by elements in } E."‘‘x is an adherent point of E if it can be approximated by elements in E."

By quantifying what this means in different ways, you can find different ways of defining an adherent point:

Proposition

Let (X,d)\p{X, d}(X,d) be a metric space and E⊆XE \subseteq XE⊆X a subset. The following are equivalent:

  1. xxx is an adherent point of EEE.
  2. x∈E‾x \in \cl{E}x∈E.
  3. x∉Ext⁡Ex \notin \Ext{E}x∈/ExtE.
  4. There exists a sequence {xn}n\set{x_n}_n{xn​}n​ of elements in EEE such that lim⁡n→∞xn=x\lim_{n\to\infty} x_n = xlimn→∞​xn​=x.
Remark.

To interpret (iii), if x∈Ext⁡Ex \in \Ext{E}x∈ExtE, then xxx is far away from EEE, so it cannot be approximated by elements in EEE. Thus, if x∉Ext⁡Ex \notin \Ext{E}x∈/ExtE, then it's close to EEE, so it's an adherent point.

Example 3.

Let (X,d)=(R,∣ ⋅ ∣)\p{X, d} = \p{\R, \abs{\:\cdot\:}}(X,d)=(R,∣⋅∣). Show that Q‾=R\cl{\Q} = \RQ​=R.
Solution.

Recall that Q\QQ is dense in R\RR: given any a<ba < ba<b, there exists an element q∈Qq \in \Qq∈Q such that a<q<ba < q < ba<q<b. Rewriting,

a<q<b  ⟺  q∈(a,b),a < q < b \iff q \in \p{a, b},a<q<b⟺q∈(a,b),

B(x,r)∩Q=(x−r,x+r)∩Q≠∅,B\p{x, r} \cap \Q = \p{x - r, x + r} \cap \Q \neq \emptyset,B(x,r)∩Q=(x−r,x+r)∩Q=∅,
