Continuity

Definition

A function f ⁣:(X,dX)→(Y,dY)\func{f}{\p{X, d_X}}{\p{Y, d_Y}}f:(X,dX​)→(Y,dY​) is continuous at x0∈Xx_0 \in Xx0​∈X if for for every ε>0\epsilon > 0ε>0, there exists δ>0\delta > 0δ>0 such that

dX(x,x0)<δ  ⟹  dY(f(x),f(x0))<ε.d_X\p{x, x_0} < \delta \implies d_Y\p{f\p{x}, f\p{x_0}} < \epsilon.dX​(x,x0​)<δ⟹dY​(f(x),f(x0​))<ε.

If fff is continuous at every x0∈Xx_0 \in Xx0​∈X, then we say that fff is continuous.
Theorem

Let f ⁣:(X,dX)→(Y,dY)\func{f}{\p{X, d_X}}{\p{Y, d_Y}}f:(X,dX​)→(Y,dY​) be a function. The following are equivalent:

  1. fff is continuous.
  2. For any x∈Xx \in Xx∈X and any sequence {xn}n\set{x_n}_n{xn​}n​ in XXX such that xn→dXxx_n \xrightarrow{d_X} xxn​dX​​x, we have f(xn)→dYf(x)f\p{x_n} \xrightarrow{d_Y} f\p{x}f(xn​)dY​​f(x).
  3. For any open set V⊆YV \subseteq YV⊆Y, the preimage f−1(V)f^{-1}\p{V}f−1(V) is open.
  4. For any closed set F⊆YF \subseteq YF⊆Y, the preimage f−1(F)f^{-1}\p{F}f−1(F) is closed.
Remark.

Note that (iii) doesn't refer to the metric at all, so in more advanced math classes, (iii) is usually taken to be the definition of continuity.

We like continuity because it lets us move limits around. For example, we can rewrite (ii) as

lim⁡n→∞f(xn)=f(lim⁡n→∞xn).\lim_{n\to\infty} f\p{x_n} = f\p{\lim_{n\to\infty} x_n}.n→∞lim​f(xn​)=f(n→∞lim​xn​).

Continuity is one of the biggest reasons why we're studying metric spaces to begin with. Before, we only had continuity defined for functions involving real numbers, but there are plenty of interesting functions defined on more general metric spaces. For example, the Fourier transform is defined on function spaces, and we can now talk about whether it's continuous or not.

Now that we have the notion of continuity, a reasonable question to ask if whether every metric space has a (non-trivial) continuous function or not. The answer is yes:

Example 1.

Let (X,d)\p{X, d}(X,d) be a (non-empty) metric space. For a fixed y∈Xy \in Xy∈X, define f ⁣:X→R\func{f}{X}{\R}f:X→R via f(x)=d(x,y)f\p{x} = d\p{x, y}f(x)=d(x,y). Show that fff is continuous.
Solution.

For this problem, we'll want to work directly from the definition. Let ε>0\epsilon > 0ε>0. Like usual, we're going to work backwards to find δ\deltaδ:

∣f(x)−f(x0)∣=∣d(x,y)−d(x,x0)∣≤d(x,x0)\abs{f\p{x} - f\p{x_0}} = \abs{d\p{x, y} - d\p{x, x_0}} \leq d\p{x, x_0}∣f(x)−f(x0​)∣=∣d(x,y)−d(x,x0​)∣≤d(x,x0​)

by the reverse triangle inequality. So, if we let δ=ε\delta = \epsilonδ=ε, then whenever d(x,x0)<δd\p{x, x_0} < \deltad(x,x0​)<δ, we have

∣f(x)−f(x0)∣≤d(x,x0)<δ=ε,\abs{f\p{x} - f\p{x_0}} \leq d\p{x, x_0} < \delta = \epsilon,∣f(x)−f(x0​)∣≤d(x,x0​)<δ=ε,

so fff is continuous.
Example 2.

Let (X,d)\p{X, d}(X,d) be a metric space with the discrete metric and (Y,d)\p{Y, d}(Y,d) be any metric space. Then every function f ⁣:X→Y\func{f}{X}{Y}f:X→Y is continuous.
Solution.

For this problem, (iii) is the easiest to prove: given any open set V⊆YV \subseteq YV⊆Y, the preimage f−1(V)f^{-1}\p{V}f−1(V) is automatically open because every set is open with the discrete metric.
Example 3.

Let

X={{an}n∈ℓ∞ | lim⁡n→∞an exists}d({an}n,{bn}n)=sup⁡n∣an−bn∣,\begin{gathered} X = \set{\set{a_n}_n \in \ell^\infty \st \lim_{n\to\infty} a_n \text{ exists}} \\ d\p{\set{a_n}_n, \set{b_n}_n} = \sup_n{\abs{a_n - b_n}}, \end{gathered}X={{an​}n​∈ℓ∞∣∣​n→∞lim​an​ exists}d({an​}n​,{bn​}n​)=nsup​∣an​−bn​∣,​

i.e., XXX is the set of bounded convergent sequences. Define f ⁣:X→R\func{f}{X}{\R}f:X→R via

f({an}n)=lim⁡n→∞an.f\p{\set{a_n}_n} = \lim_{n\to\infty} a_n.f({an​}n​)=n→∞lim​an​.

Solution.

For this problem, (iii) and (iv) aren't that useful just because it's hard to make sense of f−1(V)f^{-1}\p{V}f−1(V). So, we'll want to use an ε\epsilonε-δ\deltaδ proof.

Let ε>0\epsilon > 0ε>0. Working backwards,

∣f({an}n)−f({bn}n)∣=∣lim⁡n→∞an−lim⁡n→∞bn∣=∣lim⁡n→∞(an−bn)∣(limit law)=lim⁡n→∞∣an−bn∣(∣ ⋅ ∣ is continuous).\begin{aligned} \abs{f\p{\set{a_n}_n} - f\p{\set{b_n}_n}} &= \abs{\lim_{n\to\infty} a_n - \lim_{n\to\infty} b_n} \\ &= \abs{\lim_{n\to\infty} \p{a_n - b_n}} && \p{\text{limit law}} \\ &= \lim_{n\to\infty} \abs{a_n - b_n} && \p{\abs{\:\cdot\:}\text{ is continuous}}. \end{aligned}∣f({an​}n​)−f({bn​}n​)∣​=∣∣​n→∞lim​an​−n→∞lim​bn​∣∣​=∣∣​n→∞lim​(an​−bn​)∣∣​=n→∞lim​∣an​−bn​∣​​(limit law)(∣⋅∣ is continuous).​

∣an−bn∣≤sup⁡n∣an−bn∣=d({an}n−{bn}n),\abs{a_n - b_n} \leq \sup_n{\abs{a_n - b_n}} = d\p{\set{a_n}_n - \set{b_n}_n},∣an​−bn​∣≤nsup​∣an​−bn​∣=d({an​}n​−{bn​}n​),

∣f({an}n)−f({bn}n)∣≤d({an}n−{bn}n).\abs{f\p{\set{a_n}_n} - f\p{\set{b_n}_n}} \leq d\p{\set{a_n}_n - \set{b_n}_n}.∣f({an​}n​)−f({bn​}n​)∣≤d({an​}n​−{bn​}n​).

Compactness and Continuity

Proposition

Let f ⁣:(X,dX)→(Y,dY)\func{f}{\p{X, d_X}}{\p{Y, d_Y}}f:(X,dX​)→(Y,dY​) be continuous. Then for any compact set K⊆XK \subseteq XK⊆X, its image f(K)⊆Yf\p{K} \subseteq Yf(K)⊆Y is compact.

Usually, we say "the continuous image of a compact set is compact."

Example 4.

If f ⁣:(X,dX)→(Y,dY)\func{f}{\p{X, d_X}}{\p{Y, d_Y}}f:(X,dX​)→(Y,dY​) is a continuous bijection, then is f−1f^{-1}f−1 also continuous?
Solution.

No. For example, let f ⁣:(R,ddisc)→(R,∣ ⋅ ∣)\func{f}{\p{\R, d_{\mathrm{disc}}}}{\p{\R, \abs{\:\cdot\:}}}f:(R,ddisc​)→(R,∣⋅∣) be given by f(x)=xf\p{x} = xf(x)=x. Then fff is continuous since every function on a discrete metric space is continuous, but its inverse f−1 ⁣:(R,∣ ⋅ ∣)→(R,ddisc)\func{f^{-1}}{\p{\R, \abs{\:\cdot\:}}}{\p{\R, d_{\mathrm{disc}}}}f−1:(R,∣⋅∣)→(R,ddisc​) is not:

Let V={0}V = \set{0}V={0}, which is open since every set in a discrete metric space is open. But

(f−1)−1(V)=f({0})={0}\p{f^{-1}}^{-1}\p{V} = f\p{\set{0}} = \set{0}(f−1)−1(V)=f({0})={0}

is not open with respect to the absolute value metric. This means that f−1f^{-1}f−1 is not continuous.

However, with a few more assumptions on XXX, we can guarantee that f−1f^{-1}f−1 is automatically continuous:

Example 5.

Let f ⁣:(X,dX)→(Y,dY)\func{f}{\p{X, d_X}}{\p{Y, d_Y}}f:(X,dX​)→(Y,dY​) be a continuous function. If XXX is compact, then f−1f^{-1}f−1 is also continuous.
Solution.

We're going to prove this by using (iv). Let F⊆XF \subseteq XF⊆X be a closed set. Since XXX is compact, this means that FFF is also compact, so because fff is continuous,

(f−1)−1(F)=f(F)⊆Y\p{f^{-1}}^{-1}\p{F} = f\p{F} \subseteq Y(f−1)−1(F)=f(F)⊆Y

is also compact, hence closed. In other words, the preimage of every closed set in XXX under f−1f^{-1}f−1 is closed, so f−1f^{-1}f−1 is continuous.
