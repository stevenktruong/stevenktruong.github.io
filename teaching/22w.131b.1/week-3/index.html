Homework Comments

Limits

I noticed that a lot of students are very "limit-happy," i.e., many of you like to move limits around without justification. Starting with Homework 3, I will require that you justify any instances where you move a limit around. This is especially important if you want to swap lim⁡\limlim with max⁡\maxmax, min⁡\minmin, sup⁡\supsup, or inf⁡\infinf. (In fact, in many instances, that won't be legal.)

For example, here is a model solution:

When you take a maximum or supremum over an infinite set, you can no longer switch lim⁡\limlim and max⁡\maxmax in general:

We'll see more examples of this phenomenon later.

Cauchy Sequences and Completeness

If you remember the definition of a Cauchy sequence in R\RR, then this definition should be no surprise:

Definition

Let (X,d)\p{X, d}(X,d) be a metric space. We say a sequence {xn}n\set{x_n}_n{xn​}n​ is Cauchy if for all ε>0\epsilon > 0ε>0, there exists NNN such that d(xn,xm)<εd\p{x_n, x_m} < \epsilond(xn​,xm​)<ε whenever n,m≥Nn, m \geq Nn,m≥N.

Morally, Cauchy sequences "should" converge. In (R,∣ ⋅ ∣)\p{\R, \abs{\:\cdot\:}}(R,∣⋅∣), a sequence was convergent if and only if it is Cauchy (this is the Bolzano-Weierstrass theorem from MATH 131A). However, this is not true for general metric spaces, which is why we have an additional definition:

Definition

Let (X,d)\p{X, d}(X,d) be a metric space. If every Cauchy sequence converges in XXX, then we say that XXX is complete.
Example 3.
  1. R\RR is complete with respect to the Euclidean metric..
  2. (0,1)\p{0, 1}(0,1) is not complete with respect to the Euclidean metric.
  3. [0,1]\br{0, 1}[0,1] is complete with respect to the Euclidean metric..

Example 5.

Show that (C([0,1]),d)\p{C\p{\br{0,1}}, d}(C([0,1]),d) is complete, where ddd is the sup⁡\supsup-norm.
Solution.

As a reminder:

Let {fn}n\set{f_n}_n{fn​}n​ be a Cauchy sequence. Like in the previous example, we'll first want to find a candidate for the limit of this sequence. Notice that for a fixed x∈[0,1]x \in \br{0, 1}x∈[0,1],
