Convergence Tests

For your convenience, I'm going to restate all of the convergence tests here.

Divergence Test

Theorem (divergence test)

Let {an}\set{a_n}{an​} be a sequence. If lim⁡n→∞an\displaystyle \lim_{n\to\infty} a_nn→∞lim​an​ does not exist or if it exists and is not 000, then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ diverges.

This should always be the first thing you try. Most of the time, it'll be inconclusive, though.

Direct Comparison Test

Theorem (direct comparison test)

Suppose 0≤an≤bn0 \leq a_n \leq b_n0≤an​≤bn​ for n≥Mn \geq Mn≥M for some M>0M > 0M>0. Then:

  • If ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ diverges, then ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ diverges as well.
  • If ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges, then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges as well.

Limit Comparison Test

Theorem (limit comparison test)

Suppose an≥0a_n \geq 0an​≥0 and bn≥0b_n \geq 0bn​≥0 for n≥Mn \geq Mn≥M for some M>0M > 0M>0. Let L=lim⁡n→∞anbn\displaystyle L = \lim_{n\to\infty} \frac{a_n}{b_n}L=n→∞lim​bn​an​​.

  • If L∈(0,∞)L \in \p{0, \infty}L∈(0,∞), then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges if and only if ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges.
  • If L=0L = 0L=0 and ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges, then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges as well.
  • If L=∞L = \inftyL=∞ and ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges, then ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges as well.

The case when L=(0,∞)L = \p{0, \infty}L=(0,∞) is when this is most useful, in my experience. If L=0L = 0L=0 or L=∞L = \inftyL=∞, you can usually use direct comparison instead.

Ratio Test

Theorem (ratio test)

Let L=lim⁡n→∞∣an+1an∣\displaystyle L = \lim_{n\to\infty} \abs{\frac{a_{n+1}}{a_n}}L=n→∞lim​∣∣​an​an+1​​∣∣​.

  • If L<1L < 1L<1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ converges absolutely.
  • If L>1L > 1L>1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ diverges.
  • If L=1L = 1L=1, then the ratio test is inconclusive.

Root Test

Theorem (root test)

Let L=lim⁡n→∞∣an∣1/n\displaystyle L = \lim_{n\to\infty} \abs{a_n}^{1/n}L=n→∞lim​∣an​∣1/n.

  • If L<1L < 1L<1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ converges absolutely.
  • If L>1L > 1L>1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ diverges.
  • If L=1L = 1L=1, then the root test is inconclusive.

The ratio test and root test are pretty similar in that if one is inconclusive, the other one probably will be, too. They also "measure" how fast the terms ana_nan​ converge to 000.

  • If L<1L < 1L<1, then the terms converge fast enough that the whole series converges.

  • If L>1L > 1L>1, then the terms converge too slow so the series diverges.

  • If L=1L = 1L=1, the rate is somewhere in between too fast and too slow, which is why they're inconclusive.

Alternating Series Test

Theorem (alternating series test)

Let {an}\set{a_n}{an​} be a sequence of the form an=(−1)nbna_n = \p{-1}^n b_nan​=(−1)nbn​, where bn≥0b_n \geq 0bn​≥0, is decreasing, and lim⁡n→∞bn=0\displaystyle\lim_{n\to\infty} b_n = 0n→∞lim​bn​=0. Then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges.

Examples

The best way to really learn these tests is to practice using them.

Example 1.

Determine whether ∑n=1∞12n+3n\displaystyle \sum_{n=1}^\infty \frac{1}{2^n + 3^n}n=1∑∞​2n+3n1​ converges.
Solution.

If I remove 3n3^n3n from the denominator, I get something bigger:

12n+3n≤12n.\frac{1}{2^n + 3^n} \leq \frac{1}{2^n}.2n+3n1​≤2n1​.

All the terms are non-negative, and ∑n=1∞12n\sum_{n=1}^\infty \frac{1}{2^n}∑n=1∞​2n1​ converges because it's a geometric series with ∣r∣=12<1\abs{r} = \frac{1}{2} < 1∣r∣=21​<1. So, by direct comparison, I get that ∑n=1∞12n+3n\sum_{n=1}^\infty \frac{1}{2^n + 3^n}∑n=1∞​2n+3n1​ also converges.
Example 2.

Determine whether ∑n=1∞nn3+n2+7\displaystyle \sum_{n=1}^\infty \frac{n}{n^3 + n^2 + 7}n=1∑∞​n3+n2+7n​ converges.
Solution.

Intuitively, when nnn is very large, the main terms that matter are the leading terms:

nn3+n2+7≈nn3=1n2,\frac{n}{n^3 + n^2 + 7} \approx \frac{n}{n^3} = \frac{1}{n^2},n3+n2+7n​≈n3n​=n21​,

so intuitively, this should converge. The limit test makes this rigorous:

lim⁡n→∞nn3+n2+71n2=lim⁡n→∞n3n3+n2+7⋅1n31n3=lim⁡n→∞11+1n+7n3=1.\begin{aligned} \lim_{n\to\infty} \frac{\frac{n}{n^3 + n^2 + 7}}{\frac{1}{n^2}} &= \lim_{n\to\infty} \frac{n^3}{n^3 + n^2 + 7} \cdot \frac{\frac{1}{n^3}}{\frac{1}{n^3}} \\ &= \lim_{n\to\infty} \frac{1}{1 + \frac{1}{n} + \frac{7}{n^3}} \\ &= 1. \end{aligned}n→∞lim​n21​n3+n2+7n​​​=n→∞lim​n3+n2+7n3​⋅n31​n31​​=n→∞lim​1+n1​+n37​1​=1.​

∑n=1∞1n2\sum_{n=1}^\infty \frac{1}{n^2}∑n=1∞​n21​ converges because it's a ppp-series with p=2>1p = 2 > 1p=2>1, so by the limit comparison test, ∑n=1∞nn3+n2+7\sum_{n=1}^\infty \frac{n}{n^3 + n^2 + 7}∑n=1∞​n3+n2+7n​ converges also.
Example 3.

Determine whether ∑n=0∞22nn!\displaystyle \sum_{n=0}^\infty \frac{2^{2n}}{n!}n=0∑∞​n!22n​ converges.
Solution.

As a rule of thumb, if you ever see exponentials and/or factorials, the root or ratio test will do the trick. In this problem, the limit in the ratio test is easier to calculate:

lim⁡n→∞∣an+1an∣=lim⁡n→∞22(n+1)(n+1)!⋅n!22n=lim⁡n→∞22n+222n⋅n!(n+1)!=lim⁡n→∞22n+1=0.\begin{aligned} \lim_{n\to\infty} \abs{\frac{a_{n+1}}{a_n}} &= \lim_{n\to\infty} \frac{2^{2\p{n+1}}}{\p{n+1}!} \cdot \frac{n!}{2^{2n}} \\ &= \lim_{n\to\infty} \frac{2^{2n+2}}{2^{2n}} \cdot \frac{n!}{\p{n+1}!} \\ &= \lim_{n\to\infty} \frac{2^2}{n+1} \\ &= 0. \end{aligned}n→∞lim​∣∣​an​an+1​​∣∣​​=n→∞lim​(n+1)!22(n+1)​⋅22nn!​=n→∞lim​22n22n+2​⋅(n+1)!n!​=n→∞lim​n+122​=0.​

000 is smaller than 111, so the ratio test tells us that our series converges.
Example 4.

Determine whether ∑n=1∞(−1)n1n\displaystyle \sum_{n=1}^\infty \p{-1}^n \frac{1}{\sqrt{n}}n=1∑∞​(−1)nn​1​ converges.
Solution.

Whenever you see something like (−1)n\p{-1}^n(−1)n, chances are, you will want to use the alternating series test. To use it, there're just three things to check:

  1. 1n≥0\frac{1}{\sqrt{n}} \geq 0n​1​≥0
  2. 1n\frac{1}{\sqrt{n}}n​1​ is decreasing
  3. lim⁡n→∞1n=0\lim_{n\to\infty} \frac{1}{\sqrt{n}} = 0limn→∞​n​1​=0

These are all clear because of the properties of n\sqrt{n}n​, so by the alternating series test, the series converges.
Example 5.

Determine whether ∑n=1∞(−1)nn22n2−1\displaystyle \sum_{n=1}^\infty \p{-1}^n \frac{n^2}{2n^2 - 1}n=1∑∞​(−1)n2n2−1n2​ converges.

lim⁡n→∞(−1)n⏟divergesn22n2−1⏟converges to 12=DNE.\lim_{n\to\infty} \underbrace{\p{-1}^n}_{\text{diverges}} \underbrace{\frac{n^2}{2n^2 - 1}}_{\text{converges to } \frac{1}{2}} = \dne.n→∞lim​diverges(−1)n​​converges to 21​2n2−1n2​​​=DNE.

So, the divergence test tells us that the series diverges.
Example 6.

Determine whether ∑n=1∞sin⁡(1n2)\displaystyle \sum_{n=1}^\infty \sin\p{\frac{1}{n^2}}n=1∑∞​sin(n21​) converges.
Solution.

To solve this, you'll want to remember this limit:

lim⁡x→0sin⁡xx=1.\lim_{x\to0} \frac{\sin{x}}{x} = 1.x→0lim​xsinx​=1.

Heuristically, this tells you that if x≈0x \approx 0x≈0, then

lim⁡n→∞∣sin⁡(1n2)1n2∣=1,\lim_{n\to\infty} \abs{\frac{\sin\p{\frac{1}{n^2}}}{\frac{1}{n^2}}} = 1,n→∞lim​∣∣​n21​sin(n21​)​∣∣​=1,

and because ∑n=1∞1n2\sum_{n=1}^\infty \frac{1}{n^2}∑n=1∞​n21​ is a ppp-series with p=2>1p = 2 > 1p=2>1, it converges. Thus, by the limit comparison test, ∑n=1∞sin⁡(1n2)\sum_{n=1}^\infty \sin\p{\frac{1}{n^2}}∑n=1∞​sin(n21​) converges (absolutely).
