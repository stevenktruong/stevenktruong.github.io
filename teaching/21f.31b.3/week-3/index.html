Sequences

Definition

A sequence (an)n\p{a_n}_n(an​)n​ is an infinitely long ordered list of numbers.

When working with sequences, we're most interested in the limiting behavior, i.e., what happens as n→∞n \to \inftyn→∞.

Limits of Sequences

Definition

Let (an)n\p{a_n}_n(an​)n​ be a sequence. Then we say that (an)n\p{a_n}_n(an​)n​ converges if lim⁡n→∞an\lim_{n\to\infty} a_nlimn→∞​an​ exists, and we say that it diverges if the limit does not exist.

A lot of times, you can just replace nnn with xxx and calculate limits like you usually do:

Example 1.

Calculate lim⁡n→∞n2+13n2+4\displaystyle\lim_{n\to\infty} \frac{n^2 + 1}{3n^2 + 4}n→∞lim​3n2+4n2+1​.
Solution.

Technically, we can't use L'Hôpital's rule yet. We need differentiable functions to use it, but functions of nnn are not differentiable. This is because to take a derivative, you need to take a limit that looks like this:

f′(x)=lim⁡h→0f(x+h)−f(x)h,f'\p{x} = \lim_{h\to0} \frac{f\p{x + h} - f\p{x}}{h},f′(x)=h→0lim​hf(x+h)−f(x)​,

but sequences are only defined on whole numbers, whereas hhh can take on fractional values as h→0h \to 0h→0. However, if you replace nnn with xxx and the resulting limit exists, then it's the same as the original limit:

lim⁡n→∞n2+13n2+4=lim⁡x→∞x2+13x2+4=lim⁡x→∞2x6x=13.\lim_{n\to\infty} \frac{n^2 + 1}{3n^2 + 4} = \lim_{x\to\infty} \frac{x^2 + 1}{3x^2 + 4} = \lim_{x\to\infty} \frac{2x}{6x} = \boxed{\frac{1}{3}}.n→∞lim​3n2+4n2+1​=x→∞lim​3x2+4x2+1​=x→∞lim​6x2x​=31​​.

The bolded part of the sentence above is important: if the limit with xxx doesn't exist, then you run into trouble.

Example 2.

Calculate lim⁡n→∞sin⁡(nπ)\displaystyle\lim_{n\to\infty} \sin\p{n\pi}n→∞lim​sin(nπ).
Solution.

Let's try to replace nnn with xxx (the following thing is wrong):

lim⁡n→∞sin⁡(nπ)=lim⁡x→∞sin⁡(πx)=DNE.\lim_{n\to\infty} \sin\p{n\pi} = \lim_{x\to\infty} \sin\p{\pi x} = \dne.n→∞lim​sin(nπ)=x→∞lim​sin(πx)=DNE.

You can't write the first equals sign because the resulting limit does not exist. In this example, the limit with xxx doesn't exist, but the limit with nnn does: sin⁡(nπ)=0\sin\p{n\pi} = 0sin(nπ)=0 for every integer nnn, so

lim⁡n→∞sin⁡(nπ)=lim⁡n→∞0=0.\lim_{n\to\infty} \sin\p{n\pi} = \lim_{n\to\infty} 0 = \boxed{0}.n→∞lim​sin(nπ)=n→∞lim​0=0​.

(For this example, the fact that nnn is an integer is very important.)

Bounded Monotone Sequences

Theorem (bounded monotone sequences converge)

Let (an)n\p{a_n}_n(an​)n​ be a sequence.

  1. If (an)n\p{a_n}_n(an​)n​ is bounded above and is increasing, then (an)n\p{a_n}_n(an​)n​ converges.
  2. If (an)n\p{a_n}_n(an​)n​ is bounded below and is decreasing, then (an)n\p{a_n}_n(an​)n​ converges.

As a rule of thumb, this theorem is mainly useful when dealing with recursive sequences.

Example 3.

Let a1=4a_1 = 4a1​=4 and an=12(an−1+8)a_n = \frac{1}{2} \p{a_{n-1} + 8}an​=21​(an−1​+8) for n≥2n \geq 2n≥2. Show that (an)n\p{a_n}_n(an​)n​ converges and calculate its limit.
Solution.

To solve this problem, we'll be using induction a lot. Basically, induction says that to prove something is true for every nnn, we just need to prove two things:

  1. Base case: Prove it's true for the first case, usually n=1n = 1n=1.
  2. Inductive step: Assume it's true for n=kn = kn=k and prove it's true for n=k+1n = k + 1n=k+1 (i.e., if one case is true, then so is the next one).

We're going to try to use the theorem above to prove that this sequence converges, so we need to show that it's bounded and that it's increasing or decreasing. To figure that out, it's a good idea to write out the first couple terms and make an educated guess:

a1=4a2=12(a1+8)=6a3=12(a2+8)=7a4=12(a3+8)=7.5.\begin{aligned} a_1 &= 4 \\ a_2 &= \frac{1}{2} \p{a_1 + 8} = 6 \\ a_3 &= \frac{1}{2} \p{a_2 + 8} = 7 \\ a_4 &= \frac{1}{2} \p{a_3 + 8} = 7.5. \end{aligned}a1​a2​a3​a4​​=4=21​(a1​+8)=6=21​(a2​+8)=7=21​(a3​+8)=7.5.​

Based on these terms, we're going to make the guess that an≤10a_n \leq 10an​≤10 and that (an)n\p{a_n}_n(an​)n​ is increasing.

Proof that an≤10a_n \leq 10an​≤10:

As mentioned above, we're going to use induction here.

Base case: (n=1n = 1n=1)

a1=4≤10a_1 = 4 \leq 10a1​=4≤10, so the base case is true.

Inductive step:

To do this step, we need to assume that ak≤10a_k \leq 10ak​≤10 and then prove that ak+1≤10a_{k+1} \leq 10ak+1​≤10 also. If we use the recursive formula for ak+1a_{k+1}ak+1​, we get

ak+1=12(ak+8).a_{k+1} = \frac{1}{2} \p{a_k + 8}.ak+1​=21​(ak​+8).

Since ak≤10a_k \leq 10ak​≤10, we can replace aka_kak​ with 101010 and get something bigger overall:

ak+1=12(ak+8)≤12(10+8)=9≤10,\begin{aligned} a_{k+1} &= \frac{1}{2} \p{a_k + 8} \\ &\leq \frac{1}{2} \p{10 + 8} \\ &= 9 \\ &\leq 10, \end{aligned}ak+1​​=21​(ak​+8)≤21​(10+8)=9≤10,​

which finishes the inductive step.

Since the base case and inductive step are both true, this means that an≤10a_n \leq 10an​≤10 for every nnn.

Proof that (an)n\p{a_n}_n(an​)n​ is increasing:

(an)n\p{a_n}_n(an​)n​ increasing means that an≤an+1a_n \leq a_{n+1}an​≤an+1​ for every nnn. Like above, we're going to prove this by induction:

Base case: (n=1n = 1n=1)

a1=4a_1 = 4a1​=4 and a2=6a_2 = 6a2​=6, so a1≤a2a_1 \leq a_2a1​≤a2​, which means the base case is true.

Inductive step:

Like before, we're going to assume that ak≤ak+1a_k \leq a_{k+1}ak​≤ak+1​, and we need to prove that ak+1≤ak+2a_{k+1} \leq a_{k+2}ak+1​≤ak+2​ also. As a tip, it's almost always easier to move all the terms to one side, i.e., prove that ak+1−ak+2≤0a_{k+1} - a_{k+2} \leq 0ak+1​−ak+2​≤0 instead:

ak+1−ak+2=12(ak+8)−12(ak+1+8)=12(ak−ak+1)⏟≤0 by assumption≤0.\begin{aligned} a_{k+1} - a_{k+2} &= \frac{1}{2} \p{a_k + 8} - \frac{1}{2} \p{a_{k+1} + 8} \\ &= \frac{1}{2} \underbrace{\p{a_k - a_{k+1}}}_{\leq 0\text{ by assumption}} \\ &\leq 0. \end{aligned}ak+1​−ak+2​​=21​(ak​+8)−21​(ak+1​+8)=21​≤0 by assumption(ak​−ak+1​)​​≤0.​

This proves that ak+1≤ak+2a_{k+1} \leq a_{k+2}ak+1​≤ak+2​, so by induction, (an)n\p{a_n}_n(an​)n​ is increasing.

Finishing the problem:

We've shown that (an)n\p{a_n}_n(an​)n​ is bounded above (by 101010) and is increasing, so by the theorem, we know that the sequence converges. So, the limit L=lim⁡n→∞anL = \lim_{n\to\infty} a_nL=limn→∞​an​ exists. To complete the problem, we just need to figure out what 101010 is. We can do this via the recursive relation:

an=12(an−1+8).a_n = \frac{1}{2} \p{a_{n-1} + 8}.an​=21​(an−1​+8).

Because we know that lim⁡n→∞an\lim_{n\to\infty} a_nlimn→∞​an​ exists, we're now allowed to take limits on both sides:

lim⁡n→∞an=lim⁡n→∞12(an−1+8)=12(lim⁡n→∞an−1+8).\lim_{n\to\infty} a_n = \lim_{n\to\infty} \frac{1}{2} \p{a_{n-1} + 8} = \frac{1}{2} \p{\lim_{n\to\infty} a_{n-1} + 8}.n→∞lim​an​=n→∞lim​21​(an−1​+8)=21​(n→∞lim​an−1​+8).

The sequence (an−1)n\p{a_{n-1}}_n(an−1​)n​ is just the sequence (an)n\p{a_n}_n(an​)n​, but shifted one term over. This means that they're essentially the same sequence, which means they have the same limit, i.e., lim⁡n→∞an−1=L\lim_{n\to\infty} a_{n-1} = Llimn→∞​an−1​=L also. Thus,

L=12(L+8)  ⟹  2L=L+8  ⟹  L=8.\begin{aligned} L = \frac{1}{2} \p{L + 8} &\implies 2L = L + 8 \\ &\implies \boxed{L = 8}. \end{aligned}L=21​(L+8)​⟹2L=L+8⟹L=8​.​

Squeeze Theorem

Theorem (squeeze theorem)

Let (an)n,(bn)n,(cn)n\p{a_n}_n, \p{b_n}_n, \p{c_n}_n(an​)n​,(bn​)n​,(cn​)n​ be sequences. If:

  1. an≤bn≤cna_n \leq b_n \leq c_nan​≤bn​≤cn​
  2. lim⁡n→∞an=lim⁡n→∞cn=L\lim_{n\to\infty} a_n = \lim_{n\to\infty} c_n = Llimn→∞​an​=limn→∞​cn​=L

Then lim⁡n→∞bn=L\lim_{n\to\infty} b_n = Llimn→∞​bn​=L as well.

In general, this theorem is pretty hard to use because you have to figure out what ana_nan​ and cnc_ncn​ need to be.
