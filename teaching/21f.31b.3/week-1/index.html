Exponentials and Logarithms

Basic Properties

Definition

Let b>0b > 0b>0 and b≠1b \neq 1b=1.

  1. bxb^xbx is called an exponential.
  2. log⁡bx\log_b{x}logb​x is called a logarithm.

The reason why we're avoiding b=1b = 1b=1 is because 1x=11^x = 11x=1 for any xxx, which isn't interesting to us. Same thing for b=0b = 0b=0. Lastly, we don't want to deal with negative values of bbb because we could end up with something like (−1)1/2\p{-1}^{1/2}(−1)1/2 which is an imaginary number, which is not covered in this class.

Change-of-Base Formula

Proposition

Let a,b>0a, b > 0a,b>0 both be different from 111. Then

log⁡bx=log⁡axlog⁡ab.\log_b{x} = \frac{\log_a{x}}{\log_a{b}}.logb​x=loga​bloga​x​.
Proof.

The proof of this formula is good practice for using the properties of these functions. First, let k=log⁡bxk = \log_b{x}k=logb​x. Then because bxb^xbx and log⁡bx\log_b{x}logb​x are inverses,

bk=blog⁡bx=x.b^k = b^{\log_b{x}} = x.bk=blogb​x=x.

Then if we apply log⁡ax\log_a{x}loga​x on both sides, we get

log⁡a(bk)=log⁡ax  ⟹  klog⁡ab=log⁡ax  ⟹  k=log⁡axlog⁡ab.\begin{aligned} \log_a\p{b^k} &= \log_a{x} \\ \implies k\log_a{b} &= \log_a{x} \\ \implies k = \frac{\log_a{x}}{\log_a{b}}. \end{aligned}loga​(bk)⟹kloga​b⟹k=loga​bloga​x​.​=loga​x=loga​x

□\square□

This is how I personally remember the formula: first, I remember very roughly what it looks like:

log⁡bx=log⁡alog⁡a\log_b{x} = \frac{\log_a}{\log_a}logb​x=loga​loga​​

From here, I see that bbb is below xxx in the left-hand side, so it stays below in the end:

When b=eb = eb=e (Euler's number), a lot of really nice things happen. For example,

ddxex=ex\deriv{}{x} e^x = e^xdxd​ex=ex

eee is so important that log⁡ex\log_e{x}loge​x has its own symbol: ln⁡x\ln{x}lnx, which satisfies

ddxln⁡x=1x.\deriv{}{x} \ln{x} = \frac{1}{x}.dxd​lnx=x1​.

Integration

Since integration and differentiation are (almost) inverses, the fact that ddxex=ex\deriv{}{x} e^x = e^xdxd​ex=ex tells us that

∫ex dx=ex+C.\int e^x \,\diff{x} = e^x + C.∫exdx=ex+C.

∫bx dx=∫exln⁡b dx=∫euln⁡b du(u=xln⁡b  ⟹  du=ln⁡b dx)=euln⁡b+C=exln⁡bln⁡b+C=bxln⁡b+C.\begin{aligned} \int b^x \,\diff{x} &= \int e^{x\ln{b}} \,\diff{x} \\ &= \int \frac{e^u}{\ln{b}} \,\diff{u} && \p{u = x\ln{b} \implies \diff{u} = \ln{b} \,\diff{x}} \\ &= \frac{e^u}{\ln{b}} + C \\ &= \frac{e^{x\ln{b}}}{\ln{b}} + C \\ &= \frac{b^x}{\ln{b}} + C. \end{aligned}∫bxdx​=∫exlnbdx=∫lnbeu​du=lnbeu​+C=lnbexlnb​+C=lnbbx​+C.​(u=xlnb⟹du=lnbdx)

Logarithmic Differentiation

Which is harder to differentiate, exsin⁡xe^x\sin{x}exsinx or ex+sin⁡xe^x + \sin{x}ex+sinx? You (hopefully) said that the first one is harder, and that's because derivatives are easier with sums than with products.

The property that log⁡b(xy)=log⁡bx+log⁡by\log_b\p{xy} = \log_b{x} + \log_b{y}logb​(xy)=logb​x+logb​y is very useful for us because it turns the product into a sum, and we can use this (with the chain rule) to make some derivatives easier to compute.

Example 1.

Let f(x)=exsin⁡xf\p{x} = e^x\sin{x}f(x)=exsinx. Use logarithmic differentiation to calculate f′(x)f'\p{x}f′(x).
Solution.

Notice that

ln⁡f(x)=ln⁡(exsin⁡x)=ln⁡ex+ln⁡(sin⁡x)=x+ln⁡(sin⁡x).\begin{aligned} \ln{f\p{x}} &= \ln\p{e^x\sin{x}} \\ &= \ln{e^x} + \ln\p{\sin{x}} \\ &= x + \ln\p{\sin{x}}. \end{aligned}lnf(x)​=ln(exsinx)=lnex+ln(sinx)=x+ln(sinx).​

Then by the chain rule, taking derivatives on both sides gives

f′(x)f(x)=1+cos⁡xsin⁡x.\frac{f'\p{x}}{f\p{x}} = 1 + \frac{\cos{x}}{\sin{x}}.f(x)f′(x)​=1+sinxcosx​.

Multiplying both sides by f(x)=exsin⁡xf\p{x} = e^x\sin{x}f(x)=exsinx then gives

f′(x)=exsin⁡x+exsin⁡x⋅cos⁡xsin⁡x=exsin⁡x+excos⁡x.f'\p{x} = e^x\sin{x} + e^x\sin{x} \cdot \frac{\cos{x}}{\sin{x}} = \boxed{e^x\sin{x} + e^x\cos{x}}.f′(x)=exsinx+exsinx⋅sinxcosx​=exsinx+excosx​.
