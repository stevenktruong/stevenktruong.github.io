Inverses

What is an Inverse?

Given a function fff, we'd like to know if we can go "backwards" from fff. This is useful if you need to solve something like f(x)=yf\p{x} = yf(x)=y; if we can "undo" fff, then we automatically have the solution to the equation. This can be done if fff is invertible:

Definition

Let fff have domain DDD and range RRR. If ∃g\exists g∃g with domain RRR and range DDD such that

  • f(g(x))=xf\p{g\p{x}} = xf(g(x))=x for all x∈Rx \in Rx∈R and
  • g(f(x))=xg\p{f\p{x}} = xg(f(x))=x for all x∈Dx \in Dx∈D,

then we say fff is invertible and we call f−1=gf^{-1} = gf−1=g the inverse of fff.

When is there an Inverse?

So when does fff have an inverse? Let's look at a situation where fff does not have one.

fff doesn't have an inverse here because we can't go backwards from yyy to get both aaa and bbb. Indeed, any function ggg with domain RRR will map yyy to either aaa or bbb, but not both. The problem is that fff maps two different elements to the same thing, so if we prevent this, fff will have an inverse.

To prevent this, we need fff to map two different elements to two different things. More succinctly, a≠b  ⟹  f(a)≠f(b)a \neq b \implies f\p{a} \neq f\p{b}a=b⟹f(a)=f(b). If fff satisfies this, then we call fff one-to-one, but this isn't the only way to tell if fff is one-to-one:

Definition (one-to-one)

Let fff have domain DDD, and suppose one of the following is true about fff:

  • a≠b  ⟹  f(a)≠f(b)a \neq b \implies f\p{a} \neq f\p{b}a=b⟹f(a)=f(b) for any a,b∈Da, b \in Da,b∈D or
  • f(x)=yf\p{x} = yf(x)=y has exactly one solution for every y∈Ry \in Ry∈R or
  • f(a)=f(b)  ⟹  a=bf\p{a} = f\p{b} \implies a = bf(a)=f(b)⟹a=b for any a,b∈Da, b \in Da,b∈D or
  • fff passes the horizontal line test.

Then we say fff is one-to-one or injective.

Notice that the domain is very important when determining whether fff is one-to-one or not.

    Since fff is one-to-one, it has an inverse, which is f−1(x)=xf^{-1}\p{x} = \sqrt{x}f−1(x)=x​.

Because trig functions oscillate, there are a lot of intervals like these, so which one do we take? 000 is a nice number, so we are going to look for a domain containing 000 which doesn't change the range of our function.

There are two intervals in this case: [−π,0]\br{-\pi, 0}[−π,0] and [0,π]\br{0, \pi}[0,π]. Integration by Parts

Theorem (integration by parts)

If uuu and vvv are functions, then

∫u dv=uv−∫v du.\int u \,\diff{v} = uv - \int v \,\diff{u}.∫udv=uv−∫vdu.

Given an integral, you're free to choose uuu and dv\diff{v}dv however you like. However, the main issue after applying the formula is the integral term ∫v du\int v \,\diff{u}∫vdu, so before you start, try to think about what v duv \,\diff{u}vdu looks like.

The way I remember the formula is that you pick one thing to differentiate (uuu) and one thing to integrate (dv\diff{v}dv). Then you integrate first to get uvuvuv and then differentiate after to get v duv \,\diff{u}vdu.
