Question 2

To evaluate the integral ∫xsin⁡xcos⁡x dx\displaystyle \int x \sin{x} \cos{x} \,\diff{x}∫xsinxcosxdx using integration by parts, the most convenient choice is

  1. u=sin⁡xu = \sin{x}u=sinx, dv=xcos⁡x dx\diff{v} = x \cos{x} \,\diff{x}dv=xcosxdx
  2. u=sin⁡2xu = \sin{2x}u=sin2x, dv=x dx\diff{v} = x \,\diff{x}dv=xdx
  3. u=xsin⁡xu = x \sin{x}u=xsinx, dv=cos⁡x dx\diff{v} = \cos{x} \,\diff{x}dv=cosxdx
  4. u=xu = xu=x, dv=sin⁡2x dx\diff{v} = \sin{2x} \,\diff{x}dv=sin2xdx
  5. u=xu = xu=x, dv=cos⁡2x dx\diff{v} = \cos{2x} \,\diff{x}dv=cos2xdx

(4) is the solution.

It should (hopefully) be clear that u=xu = xu=x and du=sin⁡xcos⁡x dx\diff{u} = \sin{x} \cos{x} \,\diff{x}du=sinxcosxdx works. All you have to do then is remember that 12sin⁡2x=sin⁡xcos⁡x\frac{1}{2}\sin{2x} = \sin{x} \cos{x}21​sin2x=sinxcosx, which means that (4) is the right answer.

Question 3

For which of the following integrals will reduction formulas be needed, i.e., where substitution does not work?

  1. ∫sin⁡xcos⁡x dx\displaystyle \int \sin{x} \cos{x} \,\diff{x}∫sinxcosxdx
  2. ∫sin⁡7xcos⁡2x dx\displaystyle \int \sin^7{x} \cos^2{x} \,\diff{x}∫sin7xcos2xdx
  3. ∫sin⁡6xcos⁡2x dx\displaystyle \int \sin^6{x} \cos^2{x} \,\diff{x}∫sin6xcos2xdx
  4. ∫sin⁡5x dx\displaystyle \int \sin^5{x} \,\diff{x}∫sin5xdx
Solution.

(3) is the solution.

Substitution will always work if either the power of sin⁡x\sin{x}sinx or the power of cos⁡x\cos{x}cosx is odd, since you can use the identity sin⁡2x+cos⁡2x=1\sin^2{x} + \cos^2{x} = 1sin2x+cos2x=1 repeatedly and use u=sin⁡xu = \sin{x}u=sinx or u=cos⁡xu = \cos{x}u=cosx to get a polynomial. The only integral where the powers are both even is (3), so you would need a reduction formula for that one.
