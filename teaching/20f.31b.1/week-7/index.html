Direct Comparison

As a reminder, here's the statement of direct comparison:

Theorem (direct comparison test for integrals)

Suppose 0≤g(x)≤f(x)0 \leq g\p{x} \leq f\p{x}0≤g(x)≤f(x). Then for −∞≤a<b≤∞-\infty \leq a < b \leq \infty−∞≤a<b≤∞ (i.e., the upper and lower bounds are allowed to be infinite):

  • If ∫abg(x) dx\displaystyle \int_a^b g\p{x} \,\diff{x}∫ab​g(x)dx diverges, then ∫abf(x) dx\displaystyle \int_a^b f\p{x} \,\diff{x}∫ab​f(x)dx diverges as well.
  • If ∫abf(x) dx\displaystyle \int_a^b f\p{x} \,\diff{x}∫ab​f(x)dx converges, then ∫abg(x) dx\displaystyle \int_a^b g\p{x} \,\diff{x}∫ab​g(x)dx converges as well.

See the Week 6 Notes for some pictures and intuition for the theorem. So now that we have a good conceptual handle on comparison, how do we use it?

When approaching a problem, you should try to have an idea of whether an integral will converge or diverge (I'll talk about this more during the examples):

  • If you want to show that ∫abf(x) dx\int_a^b f\p{x} \,\diff{x}∫ab​f(x)dx converges, you need to find a larger function g(x)g\p{x}g(x).
  • If you want to show that ∫abf(x) dx\int_a^b f\p{x} \,\diff{x}∫ab​f(x)dx diverges, you need to find a smaller (but still non-negative) function g(x)g\p{x}g(x).

There are two main strategies for getting this inequalities:

Modifying f(x)f\p{x}f(x)

Here, you'll want to modify f(x)f\p{x}f(x) to get a slightly bigger or slightly smaller function. Usually, f(x)f\p{x}f(x) is a fraction, so if both the numerator and denominator are non-negative, then:

  • To get a bigger function, you can make the numerator bigger or the denominator smaller.
  • To get a smaller function, you can make the numerator smaller or the denominator bigger.
Example 1.
(8.7.74)

Determine whether ∫1∞dx(x+x2)1/3\displaystyle \int_1^\infty \frac{\diff{x}}{\p{x + x^2}^{1/3}}∫1∞​(x+x2)1/3dx​ converges or not.
Solution.

First, we need to "feel out" the integral. When xxx is really big, then x+x2≈x2x + x^2 \approx x^2x+x2≈x2, since the higher power terms will dominate. So, intuitively, we have

1(x+x2)1/3≈1(x2)1/3=1x2/3,\frac{1}{\p{x + x^2}^{1/3}} \approx \frac{1}{\p{x^2}^{1/3}} = \frac{1}{x^{2/3}},(x+x2)1/31​≈(x2)1/31​=x2/31​,

which should diverge. So, we need to find a function smaller than the integrand, which can do by making the denominator bigger. The integral is on the interval where x≥1x \geq 1x≥1, so we get x2≥xx^2 \geq xx2≥x (by multiplying by xxx on both sides of the first inequality), which means

(x+x2)1/3≤(x2+x2)1/3=(2x2)1/3=21/3x2/3.\p{x + x^2}^{1/3} \leq \p{x^2 + x^2}^{1/3} = \p{2x^2}^{1/3} = 2^{1/3} x^{2/3}.(x+x2)1/3≤(x2+x2)1/3=(2x2)1/3=21/3x2/3.

By flipping things over, we get the inequality

0≤121/3x2/3≤1(x+x2)1/3.0 \leq \frac{1}{2^{1/3} x^{2/3}} \leq \frac{1}{\p{x + x^2}^{1/3}}.0≤21/3x2/31​≤(x+x2)1/31​.

By ppp-integrals, ∫1∞dx21/3x2/3\int_1^\infty \frac{\diff{x}}{2^{1/3} x^{2/3}}∫1∞​21/3x2/3dx​ diverges, so by direct comparison, so does ∫1∞dx(x+x2)1/3\int_1^\infty \frac{\diff{x}}{\p{x + x^2}^{1/3}}∫1∞​(x+x2)1/3dx​.
Example 2.
(8.7.75)

Determine whether ∫01dxxex+x2\displaystyle \int_0^1 \frac{\diff{x}}{xe^x + x^2}∫01​xex+x2dx​ converges or not.
Solution.

Like before, we want to get a feel for the integral. When xxx is really small, x2x^2x2 doesn't really make a big impact on the integral compared to xxx, so xex+x2≈xexxe^x + x^2 \approx xe^xxex+x2≈xex (similar to how x+x2≈x2x + x^2 \approx x^2x+x2≈x2 when xxx is really big). 1xex+x2≈1xex≈1x,\frac{1}{xe^x + x^2} \approx \frac{1}{xe^x} \approx \frac{1}{x},xex+x21​≈xex1​≈x1​,

Taking Limits

It might help to brush up on what a limit means. With that said, taking limits will help us prove inequalities for large values of xxx, which is best illustrated through examples:

Example 3.
(8.7.60(a))

Show that if a>0a > 0a>0, then lim⁡x→∞xaln⁡x  ⟹  xa>2ln⁡x\displaystyle \lim_{x\to\infty} \frac{x^a}{\ln{x}} \implies x^a > 2\ln{x}x→∞lim​lnxxa​⟹xa>2lnx for xxx large enough.
Solution.

For this problem, we're already given the limit, so we just need to prove the inequality. The definition of the limit tells us that:

For all M>0, there exists N>0 such that if x≥N, then xaln⁡x>M.\text{For all $M > 0$, there exists $N > 0$ such that if $x \geq N$, then $\dfrac{x^a}{\ln{x}} > M$.}For all M>0, there exists N>0 such that if x≥N, then lnxxa​>M.

Intuitively, the limit tells us that if xxx is large enough ("... there exists N>0N > 0N>0 such that if x≥Nx \geq Nx≥N ..."), then xaln⁡x\frac{x^a}{\ln{x}}lnxxa​ becomes very big as well ("... xaln⁡x>M\frac{x^a}{\ln{x}} > Mlnxxa​>M"). The "for all M>0M > 0M>0" part tells you that you can make your function as big as you want.

The definition says something works for any M>0M > 0M>0, so it means it'll work for a specific MMM. If we set M=2M = 2M=2, then there exists N>0N > 0N>0 such that whenever x≥Nx \geq Nx≥N, we have

xaln⁡x>2  ⟹  xa>2ln⁡x.\frac{x^a}{\ln{x}} > 2 \implies x^a > 2\ln{x}.lnxxa​>2⟹xa>2lnx.

Sequences

Definition

Definition

Sequences are just infinitely long lists of numbers. For example:

1,2,3,4,5,6,7,…0,1,1,2,3,5,8,…\begin{gathered} 1, 2, 3, 4, 5, 6, 7, \ldots \\ 0, 1, 1, 2, 3, 5, 8, \ldots \end{gathered}1,2,3,4,5,6,7,…0,1,1,2,3,5,8,…​

are both sequences.

Limits

When looking at sequences, we're basically only concerned with limits as n→∞n \to \inftyn→∞. Like usual, if the limit exists, then we say that the sequence converges, and if it doesn't, we say that the sequence diverges.

The usual limit laws also apply. For example, assuming all the limits exist,

lim⁡n→∞(an+bn)=lim⁡n→∞an+lim⁡n→∞bnlim⁡n→∞(anbn)=(lim⁡n→∞an)(lim⁡n→∞bn),\begin{aligned} \lim_{n\to\infty} \p{a_n + b_n} &= \lim_{n\to\infty} a_n + \lim_{n\to\infty} b_n \\ \lim_{n\to\infty} \p{a_nb_n} &= \p{\lim_{n\to\infty} a_n} \p{\lim_{n\to\infty} b_n}, \end{aligned}n→∞lim​(an​+bn​)n→∞lim​(an​bn​)​=n→∞lim​an​+n→∞lim​bn​=(n→∞lim​an​)(n→∞lim​bn​),​

and so on. One very useful way of taking limits is replacing nnn with xxx and calculating them as usual, e.g., with L'Hôpital's rule.

Example 4.

Calculate lim⁡n→∞n2+1n2+7\displaystyle \lim_{n\to\infty} \frac{n^2 + 1}{n^2 + 7}n→∞lim​n2+7n2+1​.
Solution.
lim⁡n→∞n2+1n2+7=lim⁡x→∞x2+1x2+7=Hlim⁡x→∞2x2x=1.\begin{aligned} \lim_{n\to\infty} \frac{n^2 + 1}{n^2 + 7} &= \lim_{x\to\infty} \frac{x^2 + 1}{x^2 + 7} \\ &\overset{H}{=} \lim_{x\to\infty} \frac{2x}{2x} \\ &= 1. \end{aligned}n→∞lim​n2+7n2+1​​=x→∞lim​x2+7x2+1​=Hx→∞lim​2x2x​=1.​

Warning: If you replace nnn with xxx and the limit does not exist, then you can't make any conclusions about the original limit:

Example 5.

Calculate lim⁡n→∞sin⁡(nπ)\displaystyle \lim_{n\to\infty} \sin\p{n\pi}n→∞lim​sin(nπ).
Solution.

You may be tempted to say

lim⁡n→∞sin⁡(nπ)=lim⁡x→∞sin⁡(xπ)=DNE,\lim_{n\to\infty} \sin\p{n\pi} = \lim_{x\to\infty} \sin\p{x\pi} = \mathrm{DNE},n→∞lim​sin(nπ)=x→∞lim​sin(xπ)=DNE,

since sin⁡(xπ)\sin\p{x\pi}sin(xπ) oscillates. This is wrong because while sin⁡(xπ)\sin\p{x\pi}sin(xπ) does indeed oscillate, sin⁡(nπ)\sin\p{n\pi}sin(nπ) does not:

sin⁡(0)=0sin⁡(π)=0sin⁡(2π)=0   ⋮\begin{aligned} \sin\p{0} &= 0 \\ \sin\p{\pi} &= 0 \\ \sin\p{2\pi} &= 0 \\ &\,\,\,\vdots \end{aligned}sin(0)sin(π)sin(2π)​=0=0=0⋮​

This means that sin⁡(nπ)\sin\p{n\pi}sin(nπ) is 000 for any integer nnn, so

lim⁡n→∞sin⁡(nπ)=0.\lim_{n\to\infty} \sin\p{n\pi} = \boxed{0}.n→∞lim​sin(nπ)=0​.
