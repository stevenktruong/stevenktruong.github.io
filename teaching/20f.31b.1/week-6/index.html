Partial Fractions

Overview

You usually want to use partial fractions when you want to integrate a rational function, i.e., a function of the form

p(x)q(x),\frac{p\p{x}}{q\p{x}},q(x)p(x)​,

where ppp and qqq are polynomials. For example, the result of partial fractions looks something like

1(x−1)(x−2)=1x−2−1x−1.\frac{1}{\p{x - 1}\p{x - 2}} = \frac{1}{x - 2} - \frac{1}{x - 1}.(x−1)(x−2)1​=x−21​−x−11​.

As you can imagine, the left side is hard to integrate, but the right side is a lot easier to integrate.

Process

When setting up the partial fractions, you want to look at the factors of qqq. Here're the rules for the setup:

Linear Factors

For every factor of qqq of the form (x+a)n\p{x + a}^n(x+a)n, you need to include the following sum:

a1x+a+a2(x+a)2+⋯+an(x+a)n.\frac{a_1}{x + a} + \frac{a_2}{\p{x + a}^2} + \cdots + \frac{a_n}{\p{x + a}^n}.x+aa1​​+(x+a)2a2​​+⋯+(x+a)nan​​.

Quadratic Factors

For factors of the form (x2+ax+b)n\p{x^2 + ax + b}^n(x2+ax+b)n, you need

a1x+b1x2+ax+b+a2x+b2(x2+ax+b)2+⋯+anx+bn(x2+ax+b)n.\frac{a_1x + b_1}{x^2 + ax + b} + \frac{a_2x + b_2}{\p{x^2 + ax + b}^2} + \cdots + \frac{a_nx + b_n}{\p{x^2 + ax + b}^n}.x2+ax+ba1​x+b1​​+(x2+ax+b)2a2​x+b2​​+⋯+(x2+ax+b)nan​x+bn​​.

Once you have the setup, you need to solve for all the constants.

Improper Integrals

Introduction

You're (hopefully) familiar with the Riemann integral: for a function continuous on an interval [a,b]\br{a, b}[a,b], we can calculate its definite integral on this interval:

∫abf(x) dx.\int_a^b f\p{x} \,\diff{x}.∫ab​f(x)dx.

The idea of improper integrals is to look at integrals where fff may not be defined on all of [a,b]\br{a, b}[a,b], or integrals where b=∞b = \inftyb=∞:

Definition

Let f(x)f\p{x}f(x) be a function. If fff is not defined at aaa, then we have the following improper integral of fff:

∫abf(x) dx=lim⁡t→a+∫tbf(x) dx.\int_a^b f\p{x} \,\diff{x} = \lim_{t \to a^+} \int_t^b f\p{x} \,\diff{x}.∫ab​f(x)dx=t→a+lim​∫tb​f(x)dx.

Similarly, if fff is not defined at bbb, then

∫abf(x) dx=lim⁡t→b−∫atf(x) dx.\int_a^b f\p{x} \,\diff{x} = \lim_{t \to b^-} \int_a^t f\p{x} \,\diff{x}.∫ab​f(x)dx=t→b−lim​∫at​f(x)dx.

If b=∞b = \inftyb=∞, we also define

∫a∞f(x) dx=lim⁡t→∞∫atf(x) dx.\int_a^\infty f\p{x} \,\diff{x} = \lim_{t \to \infty} \int_a^t f\p{x} \,\diff{x}.∫a∞​f(x)dx=t→∞lim​∫at​f(x)dx.

If the limit of one of these integrals exists and is a real number, then we say that integral converges. Otherwise, in the case that the limit does not exist (including the case where the limit is ∞\infty∞), we say that the integral diverges.

In most cases, you won't be able to calculate an improper integral directly, but you'll be able to figure out if it converges or diverges (which is what the next couple weeks of the class will involve).

Pitfalls

A pretty common error when determining convergence and divergence is that you forget to check where your function is not defined. For example, the following calculation is wrong:

∫−11dxx2=−1x∣−11=−1−1=−2,\int_{-1}^1 \frac{\diff{x}}{x^2} = \left. -\frac{1}{x} \right\rvert_{-1}^1 = -1 - 1 = -2,∫−11​x2dx​=−x1​∣∣​−11​=−1−1=−2,

but we know from ppp-integrals that this should diverge, so this calculation is wrong. The reason why is because this is actually an improper integral, so we should have split it up to begin with:

∫−11dxx2=lim⁡t→0−∫−10dxx2+lim⁡t→0+∫t1dxx2,\int_{-1}^1 \frac{\diff{x}}{x^2} = \lim_{t\to0^-} \int_{-1}^0 \frac{\diff{x}}{x^2} + \lim_{t\to0^+} \int_t^1 \frac{\diff{x}}{x^2},∫−11​x2dx​=t→0−lim​∫−10​x2dx​+t→0+lim​∫t1​x2dx​,

and both integrals diverge, so the original integral must also diverge.

Direct Comparison

As a quick preview, we're going to quickly talk about direct comparison.

Theorem (direct comparison)

Suppose f(x)f\p{x}f(x) and g(x)g\p{x}g(x) are functions which satisfy 0≤f(x)≤g(x)0 \leq f\p{x} \leq g\p{x}0≤f(x)≤g(x). Then

∫abg(x) dx converges  ⟹  ∫abf(x) dx converges\int_a^b g\p{x} \,\diff{x} \text{ converges} \implies \int_a^b f\p{x} \,\diff{x} \text{ converges}∫ab​g(x)dx converges⟹∫ab​f(x)dx converges

and

∫abf(x) dx diverges  ⟹  ∫abf(x) dx diverges.\int_a^b f\p{x} \,\diff{x} \text{ diverges} \implies \int_a^b f\p{x} \,\diff{x} \text{ diverges}.∫ab​f(x)dx diverges⟹∫ab​f(x)dx diverges.

This includes the case when b=∞b = \inftyb=∞.

The picture looks something like this:

Hopefully, it's intuitive to you why this is true: if the area under ggg is finite, then the area under fff should be smaller, so it should also be finite. Similarly, if fff has infinite area, then ggg has a larger area, so it should have infinite area as well.

The assumption that everything is non-negative is important so that we can "squeeze" fff with ggg. Consider something like this:

The integrals ∫a∞f(x) dx\int_a^\infty f\p{x} \,\diff{x}∫a∞​f(x)dx and ∫a∞g(x) dx\int_a^\infty g\p{x} \,\diff{x}∫a∞​g(x)dx in this case have no relation to each other, so direct comparison does not apply in this scenario.
