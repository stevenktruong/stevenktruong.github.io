Absolute Convergence

Definition

Let {an}\set{a_n}{an​} be a sequence.

  • If ∑n=0∞∣an∣\displaystyle \sum_{n=0}^\infty \abs{a_n}n=0∑∞​∣an​∣ converges, then we say that the series is absolutely convergent.
  • If ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges, but it does not converge absolutely, then we say that the series is conditionally convergent.

Convergence Tests

For your convenience, I'm going to restate all of the convergence tests here.

Divergence Test

Theorem (divergence test)

Let {an}\set{a_n}{an​} be a sequence. If lim⁡n→∞an\displaystyle \lim_{n\to\infty} a_nn→∞lim​an​ does not exist or if it exists and is not 000, then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ diverges.

This should always be the first thing you try. Most of the time, it'll be inconclusive, though.

Integral Test

Theorem (integral test)

Let {an}\set{a_n}{an​} be a sequence and suppose that there is a function fff such that f(n)=anf\p{n} = a_nf(n)=an​ for all n≥Mn \geq Mn≥M for some M>0M > 0M>0. If f(x)f\p{x}f(x) is positive, decreasing, and continuous on [1,∞)\pco{1, \infty}[1,∞), then

∑n=1∞an converges  ⟺  ∫1∞f(x) dx converges.\sum_{n=1}^\infty a_n \text{ converges} \iff \int_1^\infty f\p{x} \,\diff{x} \text{ converges}.n=1∑∞​an​ converges⟺∫1∞​f(x)dx converges.

This one is pretty useful when it works, but as you can imagine, it can be really hard to use if your function is hard to integrate. The main application is convergence of ppp-series:

Example 1.

Determine when ∑n=1∞1np\displaystyle \sum_{n=1}^\infty \frac{1}{n^p}n=1∑∞​np1​ converges.
Solution.

Here, f(x)=x−pf\p{x} = x^{-p}f(x)=x−p, which is continuous and non-negative when x≥1x \geq 1x≥1. Also, f′(x)=−px−p−1<0f'\p{x} = -px^{-p-1} < 0f′(x)=−px−p−1<0 if x≥1x \geq 1x≥1, too, so fff is decreasing on [1,∞)\pco{1, \infty}[1,∞), which means that the integral test applies.

The improper integral ∫1∞1xp dx\int_1^\infty \frac{1}{x^p} \,\diff{x}∫1∞​xp1​dx converges if p>1p > 1p>1 and diverges if p≤1p \leq 1p≤1 by the theorem on ppp-integrals. By the integral test, this means that ∑n=1∞1np\sum_{n=1}^\infty \frac{1}{n^p}∑n=1∞​np1​ converges if and only if p>1p > 1p>1.

Direct Comparison Test

Theorem (direct comparison test)

Suppose 0≤an≤bn0 \leq a_n \leq b_n0≤an​≤bn​ for n≥Mn \geq Mn≥M for some M>0M > 0M>0. Then:

  • If ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ diverges, then ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ diverges as well.
  • If ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges, then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges as well.

Limit Comparison Test

Theorem (limit comparison test)

Suppose an≥0a_n \geq 0an​≥0 and bn≥0b_n \geq 0bn​≥0 for n≥Mn \geq Mn≥M for some M>0M > 0M>0. Let L=lim⁡n→∞anbn\displaystyle L = \lim_{n\to\infty} \frac{a_n}{b_n}L=n→∞lim​bn​an​​.

  • If L∈(0,∞)L \in \p{0, \infty}L∈(0,∞), then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges if and only if ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges.
  • If L=0L = 0L=0 and ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges, then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges as well.
  • If L=∞L = \inftyL=∞ and ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges, then ∑n=0∞bn\displaystyle \sum_{n=0}^\infty b_nn=0∑∞​bn​ converges as well.

The case when L=(0,∞)L = \p{0, \infty}L=(0,∞) is when this is most useful, in my experience. If L=0L = 0L=0 or L=∞L = \inftyL=∞, you can usually use direct comparison instead.

Example 2.

Determine whether ∑n=1∞nn3+n2+7\displaystyle \sum_{n=1}^\infty \frac{n}{n^3 + n^2 + 7}n=1∑∞​n3+n2+7n​ converges.
Solution.

Intuitively, when nnn is very large, the main terms that matter are the leading terms:

nn3+n2+7≈nn3=1n2,\frac{n}{n^3 + n^2 + 7} \approx \frac{n}{n^3} = \frac{1}{n^2},n3+n2+7n​≈n3n​=n21​,

so intuitively, this should converge. The limit test makes this rigorous:

lim⁡n→∞nn3+n2+71n2=lim⁡n→∞n3n3+n2+7⋅1n31n3=lim⁡n→∞11+1n+7n3=1.\begin{aligned} \lim_{n\to\infty} \frac{\frac{n}{n^3 + n^2 + 7}}{\frac{1}{n^2}} &= \lim_{n\to\infty} \frac{n^3}{n^3 + n^2 + 7} \cdot \frac{\frac{1}{n^3}}{\frac{1}{n^3}} \\ &= \lim_{n\to\infty} \frac{1}{1 + \frac{1}{n} + \frac{7}{n^3}} \\ &= 1. \end{aligned}n→∞lim​n21​n3+n2+7n​​​=n→∞lim​n3+n2+7n3​⋅n31​n31​​=n→∞lim​1+n1​+n37​1​=1.​

∑n=1∞1n2\sum_{n=1}^\infty \frac{1}{n^2}∑n=1∞​n21​ converges because it's a ppp-series, so by the limit comparison test, ∑n=1∞nn3+n2+7\sum_{n=1}^\infty \frac{n}{n^3 + n^2 + 7}∑n=1∞​n3+n2+7n​ converges also.

Ratio Test

Theorem (ratio test)

Let L=lim⁡n→∞∣an+1an∣\displaystyle L = \lim_{n\to\infty} \abs{\frac{a_{n+1}}{a_n}}L=n→∞lim​∣∣​an​an+1​​∣∣​.

  • If L<1L < 1L<1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ converges absolutely.
  • If L>1L > 1L>1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ diverges.
  • If L=1L = 1L=1, then the ratio test is inconclusive.

Root Test

Theorem (root test)

Let L=lim⁡n→∞∣an∣1/n\displaystyle L = \lim_{n\to\infty} \abs{a_n}^{1/n}L=n→∞lim​∣an​∣1/n.

  • If L<1L < 1L<1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ converges absolutely.
  • If L>1L > 1L>1, then ∑n=1∞an\displaystyle \sum_{n=1}^\infty a_nn=1∑∞​an​ diverges.
  • If L=1L = 1L=1, then the root test is inconclusive.

The ratio test and root test are pretty similar in that if one is inconclusive, the other one probably will be, too. They also "measure" how fast the terms ana_nan​ converge to 000.

  • If L<1L < 1L<1, then the terms converge fast enough that the whole series converges.

  • If L>1L > 1L>1, then the terms converge too slow so the series diverges.

  • If L=1L = 1L=1, the rate is somewhere in between too fast and too slow, which is why they're inconclusive.

Alternating Series Test

Theorem (alternating series test)

Let {an}\set{a_n}{an​} be a sequence of the form an=(−1)n−1bna_n = \p{-1}^{n-1}b_nan​=(−1)n−1bn​, where bn>0b_n > 0bn​>0 and bn≥bn+1b_n \geq b_{n+1}bn​≥bn+1​ for n≥1n \geq 1n≥1. Then ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ converges.

In this case, ∑n=0∞an\displaystyle \sum_{n=0}^\infty a_nn=0∑∞​an​ is called an alternating series, since the terms alternate signs at each step.

Example 3.

Determine whether ∑n=1∞(−1)nn\displaystyle \sum_{n=1}^\infty \frac{\p{-1}^n}{n}n=1∑∞​n(−1)n​ converges or not.
Solution.

We have an alternating series with bn=1nb_n = \frac{1}{n}bn​=n1​, which decreases to 000. So, by the alternating series test, ∑n=1∞(−1)nn\sum_{n=1}^\infty \frac{\p{-1}^n}{n}∑n=1∞​n(−1)n​ converges.

This is an example of a conditionally convergent series, since the harmonic series diverges.

Example 9.

Solution.
